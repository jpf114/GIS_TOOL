# 特征匹配与配准 / 角点检测

## 功能说明

提取 Harris 或 Shi-Tomasi 角点，可选导出为 JSON。

## 输入与输出

- 输入：栅格影像
- 输出：
  - 可不写文件，仅看统计
  - 若导出，建议使用 `.json`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入影像。 |
| 输出文件 | 文件路径 | 否 | 无 | 可选，导出角点 JSON。 |
| 角点方法 | 枚举 | 否 | `harris` | `harris` 或 `shi_tomasi`。 |
| 最大角点数 | 整数 | 否 | `5000` | 输出角点上限。 |
| 质量阈值 | 浮点数 | 否 | `0.01` | 角点筛选质量阈值。 |
| 最小间距 | 浮点数 | 否 | `10.0` | 角点之间的最小距离。 |
| 波段号 | 整数 | 否 | `1` | 从 1 开始。 |

## 输出内容

- 元数据返回 `corner_count` 和方法名。

## 枚举值说明

### 角点方法

- `harris`：Harris 角点
- `shi_tomasi`：Shi-Tomasi 角点

## 输出文件示例

```json
{
  "keypoints": [
    {
      "x": 210.0,
      "y": 96.0,
      "size": 3.0,
      "angle": -1.0,
      "response": 0.0
    }
  ],
  "count": 1
}
```
