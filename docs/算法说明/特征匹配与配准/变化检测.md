# 特征匹配与配准 / 变化检测

## 功能说明

对前后时相影像执行变化检测，输出变化图。

## 输入与输出

- 输入：前时相参考影像 + 后时相输入影像
- 输出：变化强度栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 参考文件 | 文件路径 | 是 | 无 | 前时相影像。 |
| 输入文件 | 文件路径 | 是 | 无 | 后时相影像。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出变化图。 |
| 变化方法 | 枚举 | 否 | `differencing` | `differencing` `ratio` `pcd`。 |
| 阈值 | 浮点数 | 否 | `0.0` | 小于等于 0 时自动按 `mean + stddev` 估阈值。 |
| 波段号 | 整数 | 否 | `1` | 从 1 开始。 |

## 注意事项

- 两幅影像尺寸必须一致。
- 输出文件是变化强度图，不是二值 mask。
- 元数据会返回变化像素数量、总像素数、变化比例和实际阈值。

## 枚举值说明

### 变化方法

- `differencing`：差值法
- `ratio`：比值法
- `pcd`：主成分差分思路
