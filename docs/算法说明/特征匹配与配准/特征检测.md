# 特征匹配与配准 / 特征检测

## 功能说明

在输入影像指定波段上提取关键点，可选导出为 JSON 文件。

## 输入与输出

- 输入：栅格影像，建议 `.tif`
- 输出：
  - 可不写输出文件，此时仅在结果中返回统计信息
  - 如果写输出文件，建议使用 `.json`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入影像路径。 |
| 输出文件 | 文件路径 | 否 | 无 | 可选，导出关键点 JSON。 |
| 方法 | 枚举 | 否 | `sift` | 可选 `sift` `orb` `akaze`。 |
| 最大特征点数 | 整数 | 否 | `5000` | 检测上限。 |
| 波段号 | 整数 | 否 | `1` | 从 1 开始。 |

## 输出内容

- JSON 文件包含关键点坐标、尺度、角度、响应值。
- 结果元数据包含 `keypoint_count` 和 `method`。

## 枚举值说明

### 方法

- `sift`：稳定性高，适合纹理较丰富场景
- `orb`：速度快，适合轻量场景
- `akaze`：速度和鲁棒性折中

## 输出文件示例

```json
{
  "keypoints": [
    {
      "x": 125.4,
      "y": 88.2,
      "size": 4.1,
      "angle": 35.0,
      "response": 0.02
    }
  ],
  "count": 1
}
```
