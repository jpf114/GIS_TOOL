# 特征匹配与配准 / ECC 配准

## 功能说明

使用 ECC 优化进行影像精配准，适合两景影像内容较接近的场景。

## 输入与输出

- 输入：参考影像 + 待配准影像
- 输出：配准后栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 参考文件 | 文件路径 | 是 | 无 | 参考影像。 |
| 输入文件 | 文件路径 | 是 | 无 | 待配准影像。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出结果。 |
| ECC 运动模型 | 枚举 | 否 | `affine` | `translation` `euclidean` `affine` `homography`。 |
| ECC 迭代次数 | 整数 | 否 | `200` | 最大迭代次数。 |
| ECC 收敛阈值 | 浮点数 | 否 | `1e-6` | 终止阈值。 |
| 重采样方式 | 枚举 | 否 | `bilinear` | `nearest` `bilinear` `cubic`。 |
| 波段号 | 整数 | 否 | `1` | 从 1 开始。 |

## 注意事项

- Debug lightweight 模式下该功能会被禁用。
- 如果输入和参考尺寸不一致，代码会先把输入缩放到参考尺寸再做 ECC。
- 元数据会返回运动模型和 warp 矩阵。

## 枚举值说明

### ECC 运动模型

- `translation`：仅平移
- `euclidean`：平移 + 旋转
- `affine`：仿射
- `homography`：单应性

### 重采样方式

- `nearest`：最近邻
- `bilinear`：双线性
- `cubic`：三次卷积
