# 特征匹配与配准 / 影像配准

## 功能说明

基于特征提取与匹配结果，将输入影像配准到参考影像空间。

## 输入与输出

- 输入：参考影像 + 待配准影像
- 输出：配准后的栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 参考文件 | 文件路径 | 是 | 无 | 目标空间参考影像。 |
| 输入文件 | 文件路径 | 是 | 无 | 待配准影像。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出配准结果。 |
| 方法 | 枚举 | 否 | `sift` | `sift` `orb` `akaze`。 |
| 匹配方法 | 枚举 | 否 | `bf` | `bf` 或 `flann`。 |
| 变换模型 | 枚举 | 否 | `affine` | `affine` `projective` `similarity` `translation`。 |
| 重采样方式 | 枚举 | 否 | `bilinear` | `nearest` `bilinear` `cubic`。 |
| 最大特征点数 | 整数 | 否 | `5000` | 特征点上限。 |
| 比率阈值 | 浮点数 | 否 | `0.75` | 匹配筛选阈值。 |
| 波段号 | 整数 | 否 | `1` | 从 1 开始。 |

## 输出说明

- 输出结果为栅格文件。
- 建议优先使用 `.tif` 保存，便于后续在 CLI、GUI 或 GIS 软件中继续处理。

## 推荐输出文件名

- `影像配准_result.tif`

## 注意事项

- 至少需要 4 个有效匹配点，否则无法估计变换。
- 输出按参考影像的空间信息写出。

## 枚举值说明

### 方法

- `sift`：稳定性高
- `orb`：速度快
- `akaze`：折中方案

### 匹配方法

- `bf`：暴力匹配
- `flann`：近似近邻匹配

### 变换模型

- `affine`：仿射变换
- `projective`：投影变换，适合透视变化
- `similarity`：相似变换
- `translation`：平移

### 重采样方式

- `nearest`：最近邻
- `bilinear`：双线性
- `cubic`：三次卷积
