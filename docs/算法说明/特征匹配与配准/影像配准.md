# 特征匹配与配准 / 影像配准

## 功能说明

基于特征提取与匹配结果，将输入影像配准到参考影像空间。

## 输入与输出

- 输入：参考影像 + 待配准影像
- 输出：配准后的栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 参考文件 | 文件路径 | 是 | 无 | 目标空间参考影像。 |
| 输入文件 | 文件路径 | 是 | 无 | 待配准影像。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出配准结果。 |
| 方法 | 枚举 | 否 | `sift` | `sift` `orb` `akaze`。 |
| 匹配方法 | 枚举 | 否 | `bf` | `bf` 或 `flann`。 |
| 变换模型 | 枚举 | 否 | `affine` | `affine` `projective` `similarity` `translation`。 |
| 重采样方式 | 枚举 | 否 | `bilinear` | `nearest` `bilinear` `cubic`。 |
| 最大特征点数 | 整数 | 否 | `5000` | 特征点上限。 |
| 比率阈值 | 浮点数 | 否 | `0.75` | 匹配筛选阈值。 |
| 波段号 | 整数 | 否 | `1` | 从 1 开始。 |

## 算法内容

影像配准通过“特征检测 -> 特征匹配 -> 估计几何变换 -> 重采样输出”的方式，把输入影像对齐到参考影像空间。当前实现支持 SIFT、ORB、AKAZE 三种特征提取方法，并支持 BF 与 FLANN 两种匹配方式。

在匹配完成后，程序会用 Lowe ratio test 去掉不稳定匹配点，只保留质量较高的对应关系。若选择 `projective`，则使用单应性矩阵；其他变换类型统一先尝试估计一般仿射矩阵，失败时再退回部分仿射矩阵。最后把原始输入波段重采样到参考影像尺寸并写出。

## 处理流程

1. 读取参考影像和输入影像的指定波段，并转换成 8 位灰度图。
2. 按 `方法` 提取两幅影像的关键点和描述子。
3. 按 `匹配方法` 对两组描述子做近邻匹配。
4. 按 `比率阈值` 执行 Lowe ratio test，筛出有效匹配点。
5. 若有效匹配少于 4 对，则返回失败。
6. 根据有效匹配点构建参考图与输入图的对应点集。
7. 按 `变换模型` 估计几何变换矩阵：
   - `projective` 使用单应性；
   - 其他模型使用仿射或部分仿射估计。
8. 按 `重采样方式` 将原始输入波段变换到参考影像尺寸。
9. 按参考影像的空间参考信息写出配准结果。

## 输出说明

- 输出结果为栅格文件。
- 建议优先使用 `.tif` 保存，便于后续在 CLI、GUI 或 GIS 软件中继续处理。

## 结果解释

- 输出结果已经对齐到参考影像的像元网格，可直接做叠加分析。
- 元数据中的 `match_count` 反映本次配准使用的有效匹配点数量。
- 如果匹配点数量勉强达到下限，虽然能输出结果，但局部几何误差可能仍然较大，建议进一步检查。

## 适用场景

- 两景影像存在平移、旋转、尺度或透视差异时的自动配准。
- 后续要做变化检测、融合分析、差值分析前的几何对齐。
- 具备明显纹理、边缘或结构信息的影像配准任务。

## 推荐输出文件名

- `影像配准_result.tif`

## 注意事项

- 至少需要 4 个有效匹配点，否则无法估计变换。
- 输出按参考影像的空间信息写出。

## 枚举值说明

### 方法

- `sift`：稳定性高
- `orb`：速度快
- `akaze`：折中方案

### 匹配方法

- `bf`：暴力匹配
- `flann`：近似近邻匹配

### 变换模型

- `affine`：仿射变换
- `projective`：投影变换，适合透视变化
- `similarity`：相似变换
- `translation`：平移

### 重采样方式

- `nearest`：最近邻
- `bilinear`：双线性
- `cubic`：三次卷积
