# 矢量数据处理 / 顶点提取

## 功能说明

提取要素顶点为点图层，并附带来源要素和顶点序号。

## 输入与输出
- 输入：矢量数据
- 输出：顶点点图层

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入矢量文件 |
| 输出文件 | 文件路径 | 是 | 无 | 输出点图层 |
| 图层 | 字符串 | 否 | 空 | 指定输入图层 |

## 输出说明

- 输出结果为矢量数据文件。
- 建议优先使用 `.gpkg` 保存，字段与编码更稳定。

## 推荐输出文件名

- `顶点提取_result.gpkg`

## 注意事项

- 适合几何检查、控制点提取和节点分析
