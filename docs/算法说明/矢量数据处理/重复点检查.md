# 矢量数据处理 / 重复点检查

## 功能说明

检查坐标完全重复的点要素并导出 CSV 结果表。

## 输入与输出
- 输入：点矢量数据
- 输出：CSV 检查结果

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入矢量文件 |
| 输出文件 | 文件路径 | 是 | 无 | 输出 CSV 文件 |
| 图层 | 字符串 | 否 | 空 | 指定输入图层 |

## 算法内容

当前实现只支持点和多点图层。程序会把每个点坐标按固定 8 位小数格式拼成键值，第一次出现的坐标只记录 FID；后续再次遇到相同键值时，就把这条记录写入结果表，作为重复点。

因此它检查的是“坐标值完全一致”的重复，不比较属性，也不做距离容差判断。多点几何在标准化后只有能转成单点对象的情况才会参与检查。

## 处理流程

1. 打开输入图层并检查几何类型是否为点或多点。
2. 创建 CSV 输出并写入表头。
3. 逐要素读取几何并做标准化。
4. 从点几何读取坐标，并格式化成坐标键值。
5. 如果该键值首次出现，则缓存当前 FID。
6. 如果该键值已存在，则输出 `source_fid,duplicate_fid,x,y`。
7. 返回重复数量和扫描数量。

## 输出说明

- 输出文件固定为 CSV。
- `source_fid` 是首次出现该坐标的要素 FID。
- `duplicate_fid` 是后续重复出现的要素 FID。
- `x,y` 是重复点的坐标值。

## 推荐输出文件名

- `重复点检查_result.csv`

## 注意事项

- 只支持点图层。
- 当前实现按 8 位小数比较，不支持距离阈值容差。
- 结果不会自动去重或删除原始点，只做检查。

## 结果解释

- 同一坐标如果出现多次，会形成多条重复记录。
- 如果结果为空，说明没有检测到完全重复坐标。
- 该工具更适合查“重复入库点”，不适合查“非常接近但不完全相同”的点。

## 适用场景

- 点位采集成果质检。
- POI、采样点、监测点等重复入库检查。
- 去重前先做重复清单输出。
