# 矢量数据处理 / 重复点检查

## 功能说明

检查坐标完全重复的点要素并导出 CSV 结果表。

## 输入与输出
- 输入：点矢量数据
- 输出：CSV 检查结果

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入矢量文件 |
| 输出文件 | 文件路径 | 是 | 无 | 输出 CSV 文件 |
| 图层 | 字符串 | 否 | 空 | 指定输入图层 |

## 输出说明

- 输出结果为矢量数据文件。
- 建议优先使用 `.gpkg` 保存，字段与编码更稳定。

## 推荐输出文件名

- `重复点检查_result.gpkg`

## 注意事项

- 适合点位质量检查和去重前分析
