# 矢量数据处理 / 面矢量化

## 功能说明

把输入栅格指定波段中连通的同值区域转换为面要素。

## 输入与输出

- 输入：栅格文件
- 输出：面矢量结果，常用 `.gpkg`、`.geojson`、`.shp`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入栅格 |
| 输出文件 | 文件路径 | 是 | 无 | 输出矢量路径 |
| 波段序号 | 整数 | 否 | `1` | 参与面矢量化的栅格波段，从 `1` 开始 |

## 算法内容

程序读取指定栅格波段，调用 GDAL 的面矢量化过程，把同一数值且相互连通的像元区域转成面要素，并在结果中写入一个整数字段 `DN` 保存原始像元值。输出图层名称固定为 `polygonized`，坐标参考继承输入栅格投影。

## 处理流程

1. 打开输入栅格并读取指定波段。
2. 根据输出文件后缀选择矢量驱动。
3. 创建输出数据集和 `polygonized` 图层。
4. 从输入栅格投影构造输出图层空间参考。
5. 创建整数字段 `DN`。
6. 调用 GDAL 面矢量化过程，把栅格区域转为面要素。
7. 统计输出面要素数量并返回结果。

## 结果解释

- 每个输出面代表一个连通的同值像元区域
- `DN` 字段记录该区域的像元值
- 输出面数量受栅格值分布和连通关系影响，通常不会等于唯一值数量

## 适用场景

- 把分类栅格转换为面图层便于矢量统计
- 从分割结果中提取对象边界
- 将二值掩膜或区域编号图转成可编辑面数据
- 作为后续融合、面积统计、拓扑检查的输入

## 输出说明

- 输出图层名称固定为 `polygonized`
- 输出字段固定包含 `DN`
- 输出坐标参考继承输入栅格投影

## 推荐输出文件名

- `polygonize_result.gpkg`

## 注意事项

- 波段序号必须对应输入栅格中实际存在的波段
- 当前实现只创建 `DN` 字段，不会附带其它栅格统计信息
- 输出格式建议优先使用 `.gpkg`，便于保存较多面要素
