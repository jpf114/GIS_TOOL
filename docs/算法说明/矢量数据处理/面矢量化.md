# 矢量数据处理 / 面矢量化

## 功能说明

将栅格指定波段转为面矢量。

## 输入与输出

- 输入：栅格文件
- 输出：面矢量文件，建议 `.gpkg` 或 `.geojson`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入栅格。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出面矢量。 |
| 波段号 | 整数 | 否 | `1` | 从 1 开始。 |

## 输出内容

- 输出图层字段中会自动写入 `DN`。

## 字段说明

- `DN`：输入栅格对应像元值。
- 同一连通区域内，若像元值一致，会被矢量化为同一面或同一组面。

## 推荐输出文件名

- `polygonized.gpkg`
- `面矢量化结果.geojson`

## 注意事项

- 当前实现按输出文件后缀判断格式：
  - `.gpkg` -> `GPKG`
  - `.shp` -> `ESRI Shapefile`
  - 其他常见情况默认按 `GeoJSON` 处理
- 推荐优先使用 `.gpkg` 或 `.geojson`。
