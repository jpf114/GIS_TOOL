# 矢量数据处理 / 并集

## 功能说明

对输入矢量和叠加矢量执行并集分析。

## 输入与输出

- 输入：
  - 输入矢量
  - 叠加矢量
- 输出：并集结果矢量，建议 `.gpkg`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 主输入矢量。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出并集结果。 |
| 图层名 | 字符串 | 否 | 无 | 主输入图层名。 |
| 叠加矢量 | 文件路径 | 是 | 无 | 第二个矢量数据。 |

## 输出说明

- 输出结果为矢量数据文件。
- 建议优先使用 `.gpkg` 保存，字段与编码更稳定。

## 推荐输出文件名

- `并集_result.gpkg`

## 注意事项

- 两个输入必须使用相同 CRS。
- 更推荐输出到 `.gpkg`。
