# 矢量数据处理 / 差集

## 功能说明

从输入图层中扣除叠加图层覆盖的部分，输出剩余几何。

## 输入与输出

- 输入：
  - 输入矢量图层
  - 叠加矢量图层
- 输出：差集结果图层，建议使用 `.gpkg`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 主输入图层 |
| 输出文件 | 文件路径 | 是 | 无 | 差集结果输出路径 |
| 图层名 | 字符串 | 否 | 无 | 输入图层名 |
| 叠加矢量 | 文件路径 | 是 | 无 | 用于扣除的图层，默认取其第一个图层 |

## 算法内容

叠加图层几何会先做标准化处理，必要时自动修复。普通情况对每个输入要素做 `Difference` 运算，得到“输入减去叠加”的剩余部分；若输入是线、叠加是面，则不直接对整条线做差，而是先按叠加面边界切分线段，再只保留位于叠加面外的线段。输出保留输入图层字段，不保留叠加图层字段。

## 处理流程

1. 打开输入图层和叠加图层。
2. 检查两者 CRS 是否一致。
3. 读取并标准化叠加图层几何。
4. 若叠加几何数量较少，预先合并；否则按输入要素局部构造候选叠加几何。
5. 创建输出图层并复制输入字段结构。
6. 逐条读取输入要素：
   - 线 + 面时先分段，再保留位于叠加范围外的线段；
   - 不相交时直接保留原几何；
   - 相交但未被完全覆盖时计算差集几何。
7. 写出非空结果。

## 结果解释

- 输出几何代表输入图层未被叠加图层覆盖的部分
- 若某个输入要素完全落在叠加图层内，则不会写出结果
- 输出字段全部来自输入图层
- 返回数量是成功写出的差集结果要素数

## 适用场景

- 从地块中扣除保护区、道路或水面范围
- 从线网中剔除位于某些区域内的线段
- 生成“剩余可用区域”或“剔除敏感区”结果

## 输出说明

- 输出图层名称固定为 `difference`
- 输出字段沿用输入图层
- 结果几何类型以输入图层有效类型为主

## 推荐输出文件名

- `difference_result.gpkg`

## 注意事项

- 输入图层与叠加图层必须使用相同 CRS
- 叠加图层中的非法几何会先尝试自动修复
- 对“线 + 面”场景，当前实现采用分段差集逻辑，而不是简单删除整条线
