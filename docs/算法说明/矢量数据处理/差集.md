# 矢量数据处理 / 差集

## 功能说明

从输入矢量中扣除叠加矢量覆盖区域。

## 输入与输出

- 输入：
  - 输入矢量
  - 叠加矢量
- 输出：差集结果矢量，建议 `.gpkg`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 主输入矢量。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出差集结果。 |
| 图层名 | 字符串 | 否 | 无 | 主输入图层名。 |
| 叠加矢量 | 文件路径 | 是 | 无 | 扣除区域矢量。 |

## 注意事项

- 两个输入必须使用相同 CRS。
