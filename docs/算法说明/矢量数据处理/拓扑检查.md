# 矢量数据处理 / 拓扑检查

## 功能说明

检查空几何、无效几何、重复面和重叠面，并导出 CSV 问题清单。

## 输入与输出
- 输入：矢量数据
- 输出：CSV 拓扑问题表

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入矢量文件 |
| 输出文件 | 文件路径 | 是 | 无 | 输出 CSV 文件 |
| 图层 | 字符串 | 否 | 空 | 指定输入图层 |

## 注意事项

- 适合作为修复前的诊断步骤
- 输出结果不会自动修改源数据
