# 矢量数据处理 / 空间连接

## 功能说明

按空间相交关系，将连接图层中的统计信息写回输入要素。

## 输入与输出

- 输入：
  - 输入矢量
  - 连接矢量
- 输出：写入连接结果后的矢量数据，建议使用 `.gpkg`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 主输入矢量文件 |
| 输出文件 | 文件路径 | 是 | 无 | 输出空间连接结果 |
| 图层名 | 字符串 | 否 | 空 | 指定输入图层名 |
| 连接图层 | 文件路径 | 是 | 无 | 用于空间连接的目标矢量文件 |
| 连接字段 | 字符串 | 否 | 空 | 可选，将匹配要素该字段值去重后写入 `join_value` |

## 输出说明

- 保留输入图层原有几何与属性字段
- 新增 `join_count` 字段，表示与当前输入要素相交的目标要素数量
- 指定 `连接字段` 时，新增 `join_value` 字段，使用 `;` 汇总匹配值
- 建议优先使用 `.gpkg` 保存，字段与编码更稳定

## 推荐输出文件名

- `空间连接_result.gpkg`

## 注意事项

- 输入图层与连接图层必须使用相同 CRS
- 当前空间关系固定为“相交”
- 当前输出保持输入要素一行一个，不展开一对多结果
