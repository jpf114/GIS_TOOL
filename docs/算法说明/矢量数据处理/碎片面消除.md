# 矢量数据处理 / 碎片面消除

## 功能说明

按最小面积阈值移除碎片多边形，输出清理后的面图层。

## 输入与输出
- 输入：面矢量数据
- 输出：清理后的面图层

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入矢量文件 |
| 输出文件 | 文件路径 | 是 | 无 | 输出矢量文件 |
| 图层 | 字符串 | 否 | 空 | 指定输入图层 |
| 最小面积 | 浮点数 | 是 | 无 | 小于该面积的碎片面将被移除 |

## 算法内容

当前实现只支持面和多面图层。程序会按 `min_area` 阈值逐部件过滤：单面如果面积小于阈值就整条删除；多面则逐个子面判断，保留面积不小于阈值的子面，移除更小的子面。

输出几何统一按多面图层写出，不会把剩余子面重新融合，也不会自动修复相邻碎片。若某个输入要素的所有子面都被删光，则该要素不会出现在输出中，并计入 `removed_feature_count`。

## 处理流程

1. 打开输入图层并检查几何类型是否为面或多面。
2. 校验 `min_area` 必须大于 0。
3. 创建输出多面图层，并复制空间参考和属性字段。
4. 逐要素读取几何并做标准化。
5. 对空几何直接跳过。
6. 对单面判断面积是否小于阈值；对多面逐子面判断。
7. 移除面积小于阈值的子面，并累计移除部件数量。
8. 如果当前要素还有剩余面部件，则复制原属性写入输出。
9. 如果当前要素所有部件都被删光，则累计移除要素数量但不输出。

## 输出说明

- 输出是多面图层。
- `feature_count` 表示最终保留下来的输出要素数。
- `removed_part_count` 表示被删掉的面部件总数。
- `removed_feature_count` 表示因所有部件都被删光而完全消失的要素数。

## 推荐输出文件名

- `碎片面消除_result.gpkg`

## 注意事项

- 只支持面图层。
- 面积阈值单位跟输入坐标系单位一致，建议在投影坐标系下使用。
- 该工具只是过滤小面，不会把碎片并到相邻大面。

## 结果解释

- 如果 `removed_part_count` 很大，说明图层里存在大量小碎片。
- `removed_feature_count` 大于 0，表示有些要素被整条删除了。
- 输出仍可能包含多部件几何，只是其中的小部件已被过滤。

## 适用场景

- 面成果清理，去掉很小的噪声面。
- 分类后处理、面矢量化后的小斑块过滤。
- 制图前做轻量几何清理。
