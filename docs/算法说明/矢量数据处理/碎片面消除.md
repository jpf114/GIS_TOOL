# 矢量数据处理 / 碎片面消除

## 功能说明

按最小面积阈值移除碎片多边形，输出清理后的面图层。

## 输入与输出
- 输入：面矢量数据
- 输出：清理后的面图层

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入矢量文件 |
| 输出文件 | 文件路径 | 是 | 无 | 输出矢量文件 |
| 图层 | 字符串 | 否 | 空 | 指定输入图层 |
| 最小面积 | 浮点数 | 是 | 无 | 小于该面积的碎片面将被移除 |

## 输出说明

- 输出结果为矢量数据文件。
- 建议优先使用 `.gpkg` 保存，字段与编码更稳定。

## 推荐输出文件名

- `碎片面消除_result.gpkg`

## 注意事项

- 面积阈值需要结合坐标系单位设置
- 建议在投影坐标系下使用
