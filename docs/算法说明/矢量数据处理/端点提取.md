# 矢量数据处理 / 端点提取

## 功能说明

提取线要素起点和终点为点图层，并附带来源要素与端点类型。

## 输入与输出
- 输入：线矢量数据
- 输出：端点点图层

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入矢量文件 |
| 输出文件 | 文件路径 | 是 | 无 | 输出点图层 |
| 图层 | 字符串 | 否 | 空 | 指定输入图层 |

## 算法内容

当前实现只支持线和多线图层。程序会逐条线提取首点和末点；如果是一条只有一个点的退化线，则只输出起点。对于多线或几何集合，会递归对每个子线分别提取端点。

输出在复制原属性的同时，会新增 `source_fid` 和 `endpoint_type` 字段。其中 `endpoint_type=0` 表示起点，`endpoint_type=1` 表示终点。

## 处理流程

1. 打开输入图层并检查几何类型是否为线或多线。
2. 创建输出点图层，复制原字段并新增 `source_fid`、`endpoint_type`。
3. 逐要素读取几何并做标准化。
4. 对空几何直接跳过。
5. 对线几何读取起点和终点；对多线递归处理每个子线。
6. 逐个端点写出到输出点图层。
7. 返回端点数量、源要素数量和跳过数量。

## 输出说明

- 输出是点图层。
- `source_fid` 表示端点来自哪条输入要素。
- `endpoint_type=0` 表示起点，`1` 表示终点。
- 多线要素可能输出多组端点。

## 推荐输出文件名

- `端点提取_result.gpkg`

## 注意事项

- 只支持线图层。
- 对多线几何，会按每个子线分别提取起终点。
- 当前实现不会自动去除重复端点。

## 结果解释

- 正常线要素通常输出 2 个端点。
- 退化线或只有一个点的线只会输出 1 个点。
- 如果多个子线共享同一端点，该坐标会在结果中重复出现。

## 适用场景

- 路网、管网起终点提取。
- 做悬挂端点检查前的辅助理解。
- 线网节点抽取与后续拓扑分析。
