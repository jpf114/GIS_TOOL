# 矢量数据处理 / 重叠检查

## 功能说明

分析同一面图层中存在面积重叠的要素对，并导出 CSV 结果表。

## 输入与输出
- 输入：面矢量数据
- 输出：CSV 检查结果

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入矢量文件 |
| 输出文件 | 文件路径 | 是 | 无 | 输出 CSV 文件 |
| 图层 | 字符串 | 否 | 空 | 指定输入图层 |

## 注意事项

- 适合做面要素质量检查
- 输出结果用于定位重叠对象，不直接修改原始数据
