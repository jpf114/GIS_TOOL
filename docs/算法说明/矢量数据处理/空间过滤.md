# 矢量数据处理 / 空间过滤

## 功能说明

按属性条件、空间范围或两者组合过滤要素，并输出新的矢量文件。

## 输入与输出

- 输入：矢量文件
- 输出：过滤结果矢量，建议 `.gpkg` 或 `.geojson`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入矢量。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出过滤结果。 |
| 图层名 | 字符串 | 否 | 无 | 留空时取默认图层。 |
| 属性过滤 | 字符串 | 否 | 无 | SQL WHERE 表达式。 |
| 空间范围 | 范围 | 否 | `0,0,0,0` | `xmin,ymin,xmax,ymax`。 |

## 注意事项

- `属性过滤` 与 `空间范围` 至少填写一个。
- 输出格式由输出后缀决定，未识别时默认走 `GeoJSON`。

## 填写示例

- 属性过滤：`population > 10000`
- 空间范围：`120.1,30.1,120.5,30.4`
- 两者同时填写时，会先应用属性过滤，再应用空间范围过滤

## 推荐输出文件名

- `filtered_result.gpkg`
- `空间过滤结果.geojson`
