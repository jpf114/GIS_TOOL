# 矢量数据处理 / 最近邻

## 功能说明

为每个输入要素寻找目标图层中的最近邻要素，并写入距离与标识。

## 输入与输出
- 输入：源矢量数据、目标矢量数据
- 输出：写入最近邻信息后的矢量数据

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入矢量文件 |
| 输出文件 | 文件路径 | 是 | 无 | 输出矢量文件 |
| 图层 | 字符串 | 否 | 空 | 指定输入图层 |
| 目标图层 | 文件路径 | 是 | 无 | 最近邻目标矢量文件 |
| 最近邻字段 | 字符串 | 否 | 空 | 目标标识字段名 |

## 输出说明

- 输出结果为矢量数据文件。
- 建议优先使用 `.gpkg` 保存，字段与编码更稳定。

## 推荐输出文件名

- `最近邻_result.gpkg`

## 注意事项

- 建议输入和目标图层使用投影坐标系
- 距离字段通常更适合米制投影场景
