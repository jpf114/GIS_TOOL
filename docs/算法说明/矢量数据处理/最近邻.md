# 矢量数据处理 / 最近邻

## 功能说明

为输入图层中的每个要素查找目标图层中距离最近的要素，并把距离与标识写回结果。

## 输入与输出

- 输入：
  - 输入矢量图层
  - 目标矢量图层
- 输出：附带最近邻信息的新矢量图层

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 主输入图层 |
| 输出文件 | 文件路径 | 是 | 无 | 结果输出路径 |
| 图层名 | 字符串 | 否 | 无 | 输入图层名 |
| 目标图层 | 文件路径 | 是 | 无 | 最近邻搜索目标图层，默认取其第一个图层 |
| 目标字段 | 字符串 | 否 | 无 | 指定时把最近邻要素该字段值写入结果 |

## 算法内容

程序采用逐要素的全量距离搜索。对每个输入要素，遍历目标图层全部要素，计算几何距离，选出距离最小的一条记录；若输入文件和目标文件是同一个数据集，还会跳过同 FID 的自身匹配。输出保留输入字段，并新增最近邻距离、最近邻 FID，以及可选的最近邻字段值。

## 处理流程

1. 打开输入图层和目标图层。
2. 检查两者 CRS 是否一致，并检查输入是否为投影坐标系。
3. 若指定目标字段，则在目标图层中查找对应字段索引。
4. 创建输出图层并复制输入字段。
5. 新增 `nearest_dist`、`nearest_fid`，以及可选的 `nearest_val`。
6. 对每个输入要素遍历目标图层全部要素，寻找最小距离。
7. 把最近邻结果写入输出要素。

## 结果解释

- `nearest_dist`：输入要素与最近目标要素之间的几何距离
- `nearest_fid`：最近目标要素的 FID
- `nearest_val`：若指定了目标字段，则写入该字段值
- 输出字段其余部分沿用输入图层

## 适用场景

- 为地块、设施、站点等要素寻找最近对象
- 给输入图层补充最近道路、最近水系、最近服务点信息
- 计算最近邻距离作为后续统计指标

## 输出说明

- 输出保留输入图层原始字段
- 额外增加最近邻结果字段
- 返回消息中的 `feature_count` 是实际处理并写出的输入要素数

## 推荐输出文件名

- `nearest_result.gpkg`

## 注意事项

- 输入图层与目标图层必须使用相同 CRS
- 当前实现要求输入图层是投影坐标系，否则距离没有意义
- 当前实现采用逐条全量搜索，没有空间索引优化，数据量大时会较慢
