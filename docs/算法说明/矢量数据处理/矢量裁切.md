# 矢量数据处理 / 矢量裁切

## 功能说明

使用裁切图层的空间范围对输入图层做几何裁切，只保留落在裁切区域内的部分。

## 输入与输出

- 输入：
  - 输入矢量图层
  - 裁切矢量图层
- 输出：裁切结果图层，建议使用 `.gpkg`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 待裁切的矢量数据 |
| 输出文件 | 文件路径 | 是 | 无 | 裁切结果输出路径 |
| 图层名 | 字符串 | 否 | 无 | 输入图层名 |
| 裁切矢量 | 文件路径 | 是 | 无 | 裁切边界数据，默认使用其第一个图层 |

## 算法内容

裁切前，程序会把裁切图层中的几何做标准化处理，必要时尝试自动修复非法几何。对于普通面面、面线、线线等情况，按相交关系计算交集；对于“线图层被面图层裁切”的情况，当前实现不是直接做一次整体交集，而是先按裁切面边界把线分段，再保留落在裁切面内的线段，以减少混合维度叠加时的异常结果。输出中只保留输入图层字段，不带裁切图层字段。

## 处理流程

1. 打开输入图层和裁切图层。
2. 检查两者是否使用相同坐标参考。
3. 读取裁切图层所有几何，并做标准化或自动修复。
4. 若裁切几何数量较少，预先合并为一个叠加几何；否则按要素局部构造候选叠加几何。
5. 创建输出图层并复制输入图层字段。
6. 逐条读取输入要素：
   - 线被面裁切时，先分段，再保留位于裁切范围内的线段；
   - 其它情况按相交关系计算交集几何。
7. 将非空结果写入输出图层。

## 结果解释

- 输出几何是输入要素被裁切后的部分
- 输出属性只来自输入图层
- 裁切边界本身不会写入结果
- 返回消息中的数量是成功写入的结果要素数

## 适用场景

- 用行政区边界裁切道路、水系、地块等矢量数据
- 从全国或全省数据中提取局部区域
- 用面范围裁切线状要素并保留有效线段
- 在专题分析前先统一分析范围

## 输出说明

- 输出图层保留输入图层字段结构
- 输出几何类型以输入图层有效类型为主
- 建议使用 `.gpkg` 保存结果，便于字段和几何稳定存储

## 推荐输出文件名

- `矢量裁切_result.gpkg`

## 注意事项

- 输入图层与裁切图层必须使用相同 CRS，否则实现会直接失败
- 裁切图层中的非法几何会先尝试自动修复；修复失败时整个操作失败
- 当前实现只使用裁切文件的第一个图层
