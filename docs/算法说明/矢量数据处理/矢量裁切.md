# 矢量数据处理 / 矢量裁切

## 功能说明

使用裁切矢量对输入矢量执行空间裁切。

## 输入与输出

- 输入：
  - 输入矢量
  - 裁切矢量
- 输出：裁切结果矢量，建议 `.gpkg`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入矢量。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出裁切结果。 |
| 图层名 | 字符串 | 否 | 无 | 输入图层名。 |
| 裁切矢量 | 文件路径 | 是 | 无 | 裁切边界矢量。 |

## 注意事项

- 输入矢量与裁切矢量必须使用相同 CRS。
- 线与面叠加时，代码额外做了线段切分处理。
