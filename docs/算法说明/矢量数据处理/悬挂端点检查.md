# 矢量数据处理 / 悬挂端点检查

## 功能说明

检查线要素未连接的端点并导出 CSV 结果表。

## 输入与输出
- 输入：线矢量数据
- 输出：CSV 检查结果

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入矢量文件 |
| 输出文件 | 文件路径 | 是 | 无 | 输出 CSV 文件 |
| 图层 | 字符串 | 否 | 空 | 指定输入图层 |

## 算法内容

当前实现只支持线和多线图层。程序会先提取所有线端点，并把端点坐标按固定 8 位小数格式做键值统计；某个端点坐标如果在全图层里只出现 1 次，就被判定为悬挂端点。

这里的连接判断是严格坐标相等判断，不考虑距离容差，也不检查线在线段中部是否相交。也就是说，它识别的是“没有其他线端点与它重合的端点”。

## 处理流程

1. 打开输入图层并检查几何类型是否为线或多线。
2. 逐要素读取几何并做标准化。
3. 为每条线提取起点和终点；多线递归提取全部子线端点。
4. 统计每个端点坐标出现的次数。
5. 创建 CSV 输出并写入表头。
6. 再遍历端点记录，把出现次数等于 1 的端点写出。
7. 输出字段为 `source_fid,endpoint_type,x,y`。

## 输出说明

- 输出文件固定为 CSV。
- `endpoint_type=0` 表示起点，`1` 表示终点。
- `x,y` 是悬挂端点坐标。
- 只有未和其他端点重合的端点才会出现在结果里。

## 推荐输出文件名

- `悬挂端点检查_result.csv`

## 注意事项

- 只支持线图层。
- 当前实现只比较端点坐标，不检查中间交叉点。
- 不支持距离容差，坐标差一点点也会被视为未连接。

## 结果解释

- 结果条数越多，说明线网中孤立端点越多。
- 如果两条线在同一点首尾相接，该点不会被记为悬挂端点。
- 如果结果为空，说明所有端点至少与另一个端点重合一次。

## 适用场景

- 路网、河网、管网的连通性初查。
- 线状成果导入前的拓扑质检。
- 查找断头路、断裂线、孤立线段。
