# 矢量数据处理 / 邻接关系

## 功能说明

分析同一面图层中要素之间的邻接关系，并导出 CSV 邻接表。

## 输入与输出
- 输入：面矢量数据
- 输出：CSV 邻接关系表

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入矢量文件 |
| 输出文件 | 文件路径 | 是 | 无 | 输出 CSV 文件 |
| 图层 | 字符串 | 否 | 空 | 指定输入图层 |

## 注意事项

- 更适合行政区、地块等面图层
- 输出结果可用于拓扑分析或分区建模
