# 矢量数据处理 / 几何属性

## 功能说明

为要素计算面积、长度、紧凑度、圆形度和走向等几何属性字段。

## 输入与输出
- 输入：矢量数据
- 输出：附加几何属性字段后的矢量数据

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入矢量文件 |
| 输出文件 | 文件路径 | 是 | 无 | 输出矢量文件 |
| 图层 | 字符串 | 否 | 空 | 指定输入图层 |

## 输出说明

- 输出结果为矢量数据文件。
- 建议优先使用 `.gpkg` 保存，字段与编码更稳定。

## 推荐输出文件名

- `几何属性_result.gpkg`

## 注意事项

- 面和线要素得到的属性字段不完全相同
- 结果适合用于后续统计或规则筛选
