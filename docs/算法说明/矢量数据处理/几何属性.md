# 矢量数据处理 / 几何属性

## 功能说明

为输入要素计算面积、长度、紧凑度、圆形度和主方向等几何指标，并把结果写入新字段。

## 输入与输出

- 输入：矢量图层
- 输出：附带几何指标字段的新矢量图层

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入矢量数据 |
| 输出文件 | 文件路径 | 是 | 无 | 结果输出路径 |
| 图层名 | 字符串 | 否 | 无 | 目标图层名 |

## 算法内容

程序复制原有图层和字段后，额外计算五个指标：`geom_area`、`geom_length`、`compactness`、`circularity`、`orientation`。其中面积、紧凑度、圆形度只对面要素有意义；长度对线要素直接计算，对面要素按边界长度计算；主方向通过采集点、线、面外环的顶点，做整体方向估计得到 `0` 到 `180` 度之间的结果。

## 处理流程

1. 打开输入图层并检查坐标参考是否为投影坐标系。
2. 创建输出图层并复制原字段。
3. 新增五个几何指标字段。
4. 逐条读取输入要素。
5. 根据几何类型分别计算面积、长度、紧凑度、圆形度。
6. 收集几何顶点并估算主方向角。
7. 把原字段和新指标一起写入结果图层。

## 结果解释

- `geom_area`：面或多面的面积；非面要素通常为 `0`
- `geom_length`：线长度；面则取边界长度；点通常为 `0`
- `compactness`：`4πA / P²`，只对面要素有意义
- `circularity`：当前实现与 `compactness` 相同
- `orientation`：按几何顶点整体分布估算的主方向角，范围约为 `0` 到 `180`

## 适用场景

- 为地块、斑块、道路等要素补充形态指标
- 做面对象紧凑度、形状规则性分析
- 在分类后处理中增加几何判别特征
- 为线面要素补充方向性信息

## 输出说明

- 输出会保留所有原始字段
- 另外新增 `geom_area`、`geom_length`、`compactness`、`circularity`、`orientation`

## 推荐输出文件名

- `geom_metrics_result.gpkg`

## 注意事项

- 当前实现要求输入图层是投影坐标系
- `compactness` 和 `circularity` 在当前实现中是同一个公式
- 点、空几何或不支持的几何类型，其部分指标可能为 `0`
