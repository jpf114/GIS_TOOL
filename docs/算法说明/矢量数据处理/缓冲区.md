# 矢量数据处理 / 缓冲区

## 功能说明

为输入要素生成指定距离的缓冲区。

## 输入与输出

- 输入：矢量文件
- 输出：缓冲区结果矢量，建议 `.gpkg`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入矢量。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出缓冲区结果。 |
| 图层名 | 字符串 | 否 | 无 | 留空时取默认图层。 |
| 距离 | 浮点数 | 否 | `100.0` | 缓冲距离，单位随数据坐标系。 |

## 输出说明

- 输出结果为矢量数据文件。
- 建议优先使用 `.gpkg` 保存，字段与编码更稳定。

## 推荐输出文件名

- `缓冲区_result.gpkg`

## 注意事项

- 输入必须是投影坐标系，地理坐标系会直接失败。
- 输出元数据会返回处理要素数、距离、输出格式等信息。
