# 矢量数据处理 / 缓冲区

## 功能说明

按给定距离为输入要素生成缓冲区，并输出新的面图层。

## 输入与输出

- 输入：点、线、面矢量图层
- 输出：缓冲区结果图层，建议使用 `.gpkg`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入矢量数据 |
| 输出文件 | 文件路径 | 是 | 无 | 缓冲区结果输出路径 |
| 图层名 | 字符串 | 否 | 无 | 目标图层名 |
| 距离 | 浮点数 | 否 | `100.0` | 缓冲距离，单位与图层坐标单位一致 |

## 算法内容

程序逐条读取输入要素几何，并调用几何对象的缓冲运算生成新几何。输出层固定按面类型创建，原字段会完整复制到结果中。对于没有几何的要素，不会生成缓冲面；结果消息中的数量表示处理过的输入要素数，不等同于输出面数量。

## 处理流程

1. 打开输入数据集并定位目标图层。
2. 检查图层坐标参考是否为投影坐标系。
3. 根据输出路径选择目标驱动并创建输出图层。
4. 输出图层名称使用原图层名加 `_buffer` 后缀。
5. 复制原字段结构。
6. 逐条读取输入要素，对其几何执行缓冲运算。
7. 将生成的缓冲面写入输出图层。
8. 返回处理数量、距离、坐标系类型、输出格式和耗时元数据。

## 结果解释

- 输出几何是缓冲后的面或多面
- 原有属性字段会保留
- `feature_count` 表示处理过的输入要素数
- `distance` 表示本次使用的缓冲距离
- `srs_type` 用于说明实现识别到的是投影坐标系还是其它类型

## 适用场景

- 为道路、河流、管线生成影响范围
- 为点位生成服务半径或警戒范围
- 为面要素外扩一定距离，构造分析区域
- 在叠加分析前先构造邻近区

## 输出说明

- 输出图层是面图层
- 字段来自输入图层，几何改为缓冲结果
- 输出格式由输出文件后缀决定，建议优先使用 `.gpkg`

## 推荐输出文件名

- `缓冲区_result.gpkg`

## 注意事项

- 当前实现要求输入图层必须是投影坐标系；地理坐标系会直接失败
- 缓冲距离按图层坐标单位解释，不会自动换算米、公里等单位
- 结果消息中的处理数量不一定等于最终写出的缓冲面数量
