# 矢量数据处理 / 栅格化

## 功能说明

将矢量图层按指定分辨率转为栅格。

## 输入与输出

- 输入：矢量文件
- 输出：栅格文件，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入矢量。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出栅格。 |
| 图层名 | 字符串 | 否 | 无 | 留空时取默认图层。 |
| 分辨率 | 浮点数 | 否 | `1.0` | 像元大小，必须大于 0。 |
| 属性字段 | 字符串 | 否 | 无 | 留空时默认烧写常数 `1`。 |

## 注意事项

- 当前实现固定输出为单波段 `Float32` GTiff。
