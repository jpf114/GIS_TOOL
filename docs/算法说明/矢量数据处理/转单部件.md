# 矢量数据处理 / 转单部件

## 功能说明

将多部件要素拆分为多个单部件要素，并保留原有属性字段。

## 输入与输出
- 输入：矢量数据
- 输出：单部件矢量数据

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入矢量文件 |
| 输出文件 | 文件路径 | 是 | 无 | 输出矢量文件 |
| 图层 | 字符串 | 否 | 空 | 指定输入图层 |

## 输出说明

- 输出结果为矢量数据文件。
- 建议优先使用 `.gpkg` 保存，字段与编码更稳定。

## 推荐输出文件名

- `转单部件_result.gpkg`

## 注意事项

- 拆分后要素数量通常会增加
