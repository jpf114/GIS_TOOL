# 矢量数据处理 / 简化

## 功能说明

按 Douglas-Peucker 算法简化线或面要素，减少顶点数量。

## 输入与输出
- 输入：矢量数据
- 输出：简化后的矢量数据

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入矢量文件 |
| 输出文件 | 文件路径 | 是 | 无 | 输出矢量文件 |
| 图层 | 字符串 | 否 | 空 | 指定输入图层 |
| 容差 | 浮点数 | 是 | 无 | 简化容差 |

## 注意事项

- 容差越大，几何形状变化越明显
- 适合发布前做轻量压缩，不适合需要高精度边界的场景
