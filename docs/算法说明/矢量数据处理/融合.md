# 矢量数据处理 / 融合

## 功能说明

按字段或整体融合相邻要素。

## 输入与输出

- 输入：矢量文件
- 输出：融合结果矢量，建议 `.gpkg`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入矢量。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出融合结果。 |
| 图层名 | 字符串 | 否 | 无 | 留空时取默认图层。 |
| 融合字段 | 字符串 | 否 | 无 | 留空时整体融合；填写时按字段值融合。 |

## 注意事项

- 融合更适合面要素。
- 输出格式由输出后缀决定，建议 `.gpkg`。
