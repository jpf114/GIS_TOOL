# 矢量数据处理 / 融合

## 功能说明

把输入图层中的多个面几何合并为更少的结果面，可按字段分组融合，也可整体融合。

## 输入与输出

- 输入：面图层
- 输出：融合结果图层，建议使用 `.gpkg`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入矢量数据 |
| 输出文件 | 文件路径 | 是 | 无 | 融合结果输出路径 |
| 图层名 | 字符串 | 否 | 无 | 输入图层名 |
| 融合字段 | 字符串 | 否 | 无 | 指定时按字段值分组融合；留空时整体融合 |

## 算法内容

程序会读取输入图层的所有几何，并先做标准化处理，必要时自动修复。若指定融合字段且字段存在，则把几何按字段值分组，每组分别做联合；若未指定字段或字段不存在，则把所有几何整体联合为一个结果。输出图层固定为多面类型，只在按字段融合时保留该一个分组字段，不复制原始全部属性。

## 处理流程

1. 打开输入图层并检查坐标参考是否为投影坐标系。
2. 遍历所有要素，读取并标准化几何。
3. 若指定融合字段且字段存在，则按字段值分组；否则收集到一个总组中。
4. 创建输出图层，几何类型固定为 `MultiPolygon`。
5. 若按字段融合，则创建对应字段。
6. 对每个分组执行几何联合。
7. 将每个非空联合结果写出。

## 结果解释

- 未指定融合字段时，通常只得到一个融合后的总结果
- 指定融合字段时，每个字段值对应一个融合结果
- 输出字段并不会完整保留原图层，只保留分组字段或不保留属性
- 返回数量是最终写出的融合结果数量，不是输入要素数

## 适用场景

- 按行政区、类别等字段合并相邻面
- 把碎片化面要素合成为更完整的分析单元
- 在面级统计前先减少要素数量

## 输出说明

- 输出图层名称固定为 `dissolved`
- 输出几何固定为 `MultiPolygon`
- 指定融合字段时只保留该字段

## 推荐输出文件名

- `dissolve_result.gpkg`

## 注意事项

- 当前实现要求输入图层是投影坐标系
- 若填写了融合字段但字段不存在，当前实现不会报错，而是退化为整体融合
- 输出不保留原始全部属性字段，这是当前实现与很多桌面 GIS 默认行为的差异
