# 矢量数据处理 / 中点提取

## 功能说明

提取线要素几何中点为点图层，并附带来源要素。

## 输入与输出
- 输入：线矢量数据
- 输出：中点点图层

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入矢量文件 |
| 输出文件 | 文件路径 | 是 | 无 | 输出点图层 |
| 图层 | 字符串 | 否 | 空 | 指定输入图层 |

## 算法内容

当前实现只支持线和多线图层。它提取的不是顶点序列的中间节点，而是沿线长度累计到 50% 位置时的几何中点。对于折线，会逐段累计长度并在线段内插值，所以结果反映的是路径长度一半处的位置。

对总长度为 0 的退化线，当前实现会直接使用起点作为中点结果。对多线和几何集合，会对每个子线分别生成一个中点。

## 处理流程

1. 打开输入图层并检查几何类型是否为线或多线。
2. 创建输出点图层，复制原字段并新增 `source_fid`。
3. 逐要素读取几何并做标准化。
4. 对空几何直接跳过。
5. 对每条线计算总长度。
6. 沿线逐段累计长度，定位到总长度 50% 所在的线段。
7. 在目标线段内按比例插值生成中点，并写出结果。

## 输出说明

- 输出是点图层。
- `source_fid` 表示中点所属输入要素。
- 一条普通线通常输出一个中点；多线会按子线分别输出。
- 中点位置按几何长度计算，不按顶点个数平均。

## 推荐输出文件名

- `中点提取_result.gpkg`

## 注意事项

- 只支持线图层。
- 折线的中点可能不在某个已有顶点上，而在线段内部。
- 总长度为 0 的退化线会回退到起点。

## 结果解释

- 输出点表示沿线走到一半长度时的位置。
- 对形状弯曲明显的线，中点位置可能和视觉上“图形中心”不同。
- 多线结果中，同一输入要素可能对应多个中点。

## 适用场景

- 道路线、河流线等的标注落点生成。
- 线性设施中段采样。
- 后续做线中部属性挂接或展示。
