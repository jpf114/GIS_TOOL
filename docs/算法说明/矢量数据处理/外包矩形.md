# 矢量数据处理 / 外包矩形

## 功能说明

按要素生成外包矩形面，并保留原有属性字段。

## 输入与输出
- 输入：矢量数据
- 输出：外包矩形图层

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入矢量文件 |
| 输出文件 | 文件路径 | 是 | 无 | 输出矢量文件 |
| 图层 | 字符串 | 否 | 空 | 指定输入图层 |

## 输出说明

- 输出结果为矢量数据文件。
- 建议优先使用 `.gpkg` 保存，字段与编码更稳定。

## 推荐输出文件名

- `外包矩形_result.gpkg`

## 注意事项

- 适合做快速范围索引或概要表达
