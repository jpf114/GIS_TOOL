# 矢量数据处理 / 外包矩形

## 功能说明

按要素生成外包矩形面，并保留原有属性字段。

## 输入与输出
- 输入：矢量数据
- 输出：外包矩形面图层

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入矢量文件 |
| 输出文件 | 文件路径 | 是 | 无 | 输出矢量文件 |
| 图层 | 字符串 | 否 | 空 | 指定输入图层 |

## 算法内容

当前实现按单个要素计算最小轴对齐外包矩形。程序会读取每条要素的标准化几何，取其包络框 `MinX/MinY/MaxX/MaxY`，再根据四个角点构造一个矩形面。

这里的外包矩形是坐标轴对齐矩形，不会计算最小旋转外接矩形，也不会随原几何方向旋转。

## 处理流程

1. 打开输入图层并创建输出面图层。
2. 复制输入图层的坐标系和属性字段。
3. 逐要素读取几何并做标准化。
4. 对空几何直接跳过。
5. 读取几何包络框范围。
6. 按包络框四角构造闭合矩形面。
7. 复制原属性字段并写入输出图层。

## 输出说明

- 输出是面图层。
- 每个输入要素对应一个轴对齐矩形面。
- 属性字段保留，几何改成外包矩形。
- 空几何不会输出。

## 推荐输出文件名

- `外包矩形_result.gpkg`

## 注意事项

- 结果不是旋转最小矩形，而是普通包络框。
- 对细长斜向目标，矩形面积可能明显大于原几何面积。
- 输出坐标系与输入保持一致。

## 结果解释

- 输出矩形覆盖原几何的全部范围。
- 矩形面积越大于原始面积，说明原要素越不规则或方向越偏斜。
- `skipped_count` 反映被忽略的空几何数量。

## 适用场景

- 快速生成检索框、裁切框、最小显示范围。
- 做粗粒度空间索引或预筛选范围。
- 后续栅格裁切、目标概览显示。
