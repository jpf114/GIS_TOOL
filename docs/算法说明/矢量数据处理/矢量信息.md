# 矢量数据处理 / 矢量信息

## 功能说明

读取矢量数据集的结构信息，输出驱动、图层、几何类型、范围、坐标参考、要素数量和字段定义等文本结果。

## 输入与输出

- 输入：矢量文件，常见如 `.shp`、`.gpkg`、`.geojson`
- 输出：不生成结果文件，直接返回文本信息

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 要读取的矢量数据集 |
| 图层名 | 字符串 | 否 | 无 | 指定时只输出匹配图层的信息 |

## 算法内容

该功能不做空间分析，而是按数据集结构逐层读取元信息。程序先打开矢量数据集，再遍历其中的图层；对每个需要输出的图层，继续读取几何类型、范围、空间参考、要素数量和字段定义。若坐标参考能识别出权威代码，则输出类似 `EPSG:xxxx` 的形式；否则输出截断后的 WKT 文本；若图层没有坐标参考，则明确标记为 `(none)`。

## 处理流程

1. 打开输入矢量数据集。
2. 输出数据集驱动名称和图层总数。
3. 逐个遍历图层；若填写了图层名，则仅输出同名图层。
4. 对目标图层读取几何类型。
5. 尝试读取图层范围。
6. 读取空间参考，优先输出权威代码，其次输出截断后的 WKT。
7. 读取要素总数。
8. 读取字段总数，并逐项输出字段名与字段类型。

## 结果解释

- `Driver`：数据集驱动类型，如 `ESRI Shapefile`、`GPKG`
- `Layers`：数据集中的图层数量
- `Layer [n]`：第 `n` 个图层的详细信息
- `Geometry`：图层几何类型，如 `Point`、`Polygon`、`MultiPolygon`
- `Extent`：图层范围，格式为 `xmin,ymin - xmax,ymax`
- `CRS`：图层坐标参考；可能是 `EPSG` 编码、截断的 WKT，或 `(none)`
- `Features`：图层要素数量
- `Fields`：字段数量；其后会列出字段名和字段类型

## 适用场景

- 接手外部矢量数据时先确认图层结构
- 检查字段名和字段类型是否满足后续处理要求
- 快速确认图层范围和坐标参考是否正确
- 在批处理前核对数据集中是否包含目标图层

## 输出说明

- 本功能不写出新文件
- 结果以文本形式返回，适合命令行查看或作为后续判断依据

## 推荐输出文件名

- 无

## 注意事项

- 图层名参数只影响输出范围，不会修改原始数据
- 若图层没有权威坐标编码，返回的 CRS 可能是截断后的 WKT 文本
- 若指定的图层名没有匹配到实现中的实际图层，当前实现不会报错，只会缺少该图层明细
