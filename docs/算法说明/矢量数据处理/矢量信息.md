# 矢量数据处理 / 矢量信息

## 功能说明

查看矢量数据集中的图层、要素数、字段与空间参考信息。

## 输入与输出

- 输入：矢量文件，常用 `.shp` `.gpkg` `.geojson`
- 输出：无文件输出，仅返回文本结果

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入矢量。 |
| 图层名 | 字符串 | 否 | 无 | 留空时通常取第一个图层。 |

## 返回内容

- 数据驱动类型
- 图层数量
- 每个图层的几何类型
- 图层范围
- CRS
- 要素数量
- 字段数量与字段名、字段类型

## 返回内容说明

- `Driver`：矢量驱动，如 `ESRI Shapefile`、`GPKG`
- `Layers`：数据集内图层数
- `Layer [n]`：第 n 个图层的信息
- `Geometry`：几何类型，如 `Polygon`、`MultiPolygon`
- `Extent`：图层范围
- `CRS`：若可识别则显示如 `EPSG:4490`
- `Features`：要素总数
- `Fields`：字段总数，下面会逐项列出字段名和字段类型

## 注意事项

- 如果填写了 `图层名`，结果只显示该图层。
- 本功能不生成输出文件。
