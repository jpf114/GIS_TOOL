# 分类统计 / SVM 分类

## 功能说明

使用训练样本 CSV 对多波段栅格执行监督分类，当前采用 OpenCV SVM 的 RBF 核实现。

## 输入与输出

- 输入：
  - 输入栅格：建议 `.tif`
  - 训练样本 CSV：至少包含一个标签列和若干特征列
- 输出：
  - 分类结果栅格：建议 `.tif`

## 训练样本 CSV 格式

典型格式如下：

```csv
label,b1,b2
1,10,20
1,12,18
2,200,180
2,195,185
```

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 待分类栅格 |
| 输出文件 | 文件路径 | 是 | 无 | 输出分类结果 |
| 训练样本 CSV | 文件路径 | 是 | 无 | 监督分类样本文件 |
| 标签列 | 字符串 | 否 | `label` | 类别标签列名 |
| 波段列表 | 字符串 | 否 | 空 | 逗号分隔，留空表示使用全部波段 |

## 输出说明

- 输出结果为单波段分类栅格
- 像元值即训练样本中的类别编号

## 推荐输出文件名

- `svm_classify_result.tif`

## 注意事项

- 特征列数量必须与 `bands` 数量一致
- 训练样本至少应覆盖两个类别
