# 分类统计 / 最大似然分类

## 功能说明

使用训练样本 CSV 估计各类别均值和协方差，对多波段栅格执行最大似然监督分类。

## 输入与输出

- 输入：
  - 输入栅格：建议 `.tif`
  - 训练样本 CSV：至少包含一个标签列和若干特征列
- 输出：
  - 分类结果栅格：建议 `.tif`

## 训练样本 CSV 格式

```csv
label,b1,b2
1,10,20
1,12,18
2,200,180
2,195,185
```

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 待分类栅格 |
| 输出文件 | 文件路径 | 是 | 无 | 输出分类结果 |
| 训练样本 CSV | 文件路径 | 是 | 无 | 监督分类样本文件 |
| 标签列 | 字符串 | 否 | `label` | 类别标签列名 |
| 波段列表 | 字符串 | 否 | 空 | 逗号分隔，留空表示使用全部波段 |

## 算法内容

最大似然分类是一种经典监督分类方法。它假设每个类别在特征空间中都可以用一组统计参数来描述，核心是根据训练样本估计各类别的均值和协方差，再判断每个像元“最像”哪个类别。当前实现按训练样本逐类估计统计参数，并对整幅影像逐像元计算各类别得分，最终取最大得分类别作为输出。

和另外两种监督分类一样，这里的训练输入是 CSV 特征表。每一行样本都要提供类别标签与对应特征值，算法不会自己去空间数据里抽样。

## 处理流程

1. 读取训练样本 CSV，识别标签列与特征列。
2. 解析样本，将不同类别的训练记录分别归组。
3. 确定参与分类的波段列表；如果没有指定，则默认使用输入栅格全部波段。
4. 检查训练样本特征列数量与波段数量是否一致。
5. 对每个类别分别计算样本均值和协方差矩阵，并生成后续判别所需的统计参数。
6. 读取输入栅格中参与分类的各个波段。
7. 逐像元提取波段值，形成待判别特征向量。
8. 将该特征向量与每个类别的统计参数逐一比较，计算各类别得分。
9. 选取得分最高的类别作为该像元分类结果，并写入单波段输出栅格。

## 输出说明

- 输出结果为单波段分类栅格
- 像元值即最大后验得分类别编号

## 结果解释

- 输出像元值表示该像元在当前统计模型下最可能属于哪个类别。
- 该方法依赖训练样本统计特征是否稳定；如果某类样本过少、方差异常或类别高度重叠，结果容易不稳定。
- 由于当前实现需要估计协方差矩阵，因此样本数量过小或特征维度偏高时，分类效果往往不如样本更充分时稳定。

## 适用场景

- 希望采用传统统计判别方式完成监督分类。
- 各类别样本分布相对稳定，且已有较规范的训练样本表。
- 用于与 SVM、随机森林做分类效果对比。

## 推荐输出文件名

- `max_likelihood_classify_result.tif`

## 注意事项

- 至少需要两个类别
- 样本过少时统计参数会较不稳定
