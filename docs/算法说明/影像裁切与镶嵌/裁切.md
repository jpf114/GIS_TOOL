# 影像裁切与镶嵌 / 裁切

## 功能说明

按矩形范围或裁切矢量对输入栅格进行裁切。

## 输入与输出

- 输入：
  - 栅格：建议 `.tif`
  - 裁切矢量：常用 `.shp` `.gpkg` `.geojson`
- 输出：裁切后的栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入栅格路径。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出裁切结果。 |
| 空间范围 | 范围 | 否 | `0,0,0,0` | 格式 `xmin,ymin,xmax,ymax`。 |
| 裁切矢量 | 文件路径 | 否 | 无 | 用于裁切的矢量边界。 |

## 处理逻辑

- `空间范围` 和 `裁切矢量` 至少要填一个。
- 如果提供 `裁切矢量`，优先走按矢量裁切。
- 如果只提供 `空间范围`，走矩形窗口裁切。

## 填写示例

- 空间范围：`500000,3200000,510000,3210000`
- 裁切矢量：`D:/data/clip_boundary.gpkg`
- 推荐输出文件：`D:/out/clipped.tif`

## 注意事项

- 输出一般建议使用 `.tif`。
- `空间范围` 默认全零时会被视为未填写。
