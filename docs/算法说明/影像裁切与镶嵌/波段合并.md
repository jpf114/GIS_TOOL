# 影像裁切与镶嵌 / 波段合并

## 功能说明

将多个单波段栅格按顺序合并成一个多波段栅格。

## 输入与输出

- 输入：多个单波段栅格
- 输出：多波段栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 否 | 无 | 可传逗号分隔文件列表。 |
| 波段列表 | 字符串 | 否 | 无 | 额外文件列表，逗号分隔。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出多波段栅格。 |

## 填写示例

### 方式一：全部写在 `输入文件`

```text
D:/data/b1.tif,D:/data/b2.tif,D:/data/b3.tif
```

### 方式二：`输入文件` + `波段列表`

输入文件：

```text
D:/data/b1.tif
```

波段列表：

```text
D:/data/b2.tif,D:/data/b3.tif
```

### 输出说明

- 输出结果为栅格文件。
- 建议优先使用 `.tif` 保存，便于后续在 CLI、GUI 或 GIS 软件中继续处理。

## 推荐输出文件名

```text
D:/data/b1_cutting_merge_bands.tif
```

## 处理逻辑

- `输入文件` 和 `波段列表` 至少填一个。
- 两者会被拼接成统一输入列表，再按顺序合并。

## 注意事项

- 输出建议使用 `.tif`。
