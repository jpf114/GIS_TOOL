# 影像裁切与镶嵌 / 分块

## 功能说明

按固定像素块大小将输入栅格切分成多个小图块。

## 输入与输出

- 输入：单个栅格文件
- 输出：目录输出，内部自动生成多个 `.tif` 图块

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入栅格路径。 |
| 输出文件 | 文件路径 | 是 | 无 | 实际上这里应理解为输出目录路径。 |
| 分块大小 | 整数 | 否 | `1024` | 每块边长，单位像素。 |
| 重叠像素 | 整数 | 否 | `0` | 相邻块之间的重叠像素数。 |

## 算法内容

分块用于把一幅大栅格按固定像素窗口切成多个小图块。当前实现以固定块大小逐行逐列遍历整幅影像，并对每个块调用栅格窗口裁切流程输出单独文件。可选重叠像素用于让相邻图块之间保留边界冗余。

## 处理流程

1. 打开输入栅格，读取宽度、高度和波段数。
2. 检查输出目录是否存在，不存在则自动创建。
3. 按 `tile_size` 从左到右、从上到下遍历整幅影像。
4. 对每个块计算其左上角偏移、实际宽高以及可选重叠范围。
5. 按 `srcwin` 参数调用 GDALTranslate 截取当前窗口。
6. 按 `tile_x_y.tif` 规则生成图块文件名。
7. 循环直到整幅影像被分割完毕。

## 处理逻辑

- 如果输出目录不存在，代码会自动创建。
- 输出文件命名形如 `tile_x_y.tif`。

## 结果解释

- 输出不是单个文件，而是一组按行列编号命名的小图块。
- 设置重叠像素后，相邻图块边缘会包含重复区域，便于后续拼接或边缘处理。

## 适用场景

- 大影像切块处理、模型推理前分片和分布式任务准备。

## 输出说明

- 输出为目录，其中会自动生成多个 `.tif` 图块文件。
- 图块名称按 `tile_x_y.tif` 规则生成。

## 推荐输出文件名

- `tiles/`

## 注意事项

- GUI 当前参数名叫“输出文件”，但代码真实语义更接近“输出目录”。
