# 影像裁切与镶嵌 / 分块

## 功能说明

按固定像素块大小将输入栅格切分成多个小图块。

## 输入与输出

- 输入：单个栅格文件
- 输出：目录输出，内部自动生成多个 `.tif` 图块

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入栅格路径。 |
| 输出文件 | 文件路径 | 是 | 无 | 实际上这里应理解为输出目录路径。 |
| 分块大小 | 整数 | 否 | `1024` | 每块边长，单位像素。 |
| 重叠像素 | 整数 | 否 | `0` | 相邻块之间的重叠像素数。 |

## 处理逻辑

- 如果输出目录不存在，代码会自动创建。
- 输出文件命名形如 `tile_x_y.tif`。

## 输出说明

- 输出为目录，其中会自动生成多个 `.tif` 图块文件。
- 图块名称按 `tile_x_y.tif` 规则生成。

## 推荐输出文件名

- `tiles/`

## 注意事项

- GUI 当前参数名叫“输出文件”，但代码真实语义更接近“输出目录”。
