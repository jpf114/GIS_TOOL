# 影像处理与分析 / K-Means 分割

## 功能说明

按所有波段特征对整幅影像做 K-Means 聚类分割。

## 输入与输出

- 输入：多波段或单波段栅格
- 输出：聚类标签栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入影像。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出聚类结果。 |
| 波段号 | 整数 | 否 | `1` | GUI 会显示，但当前实现实际按全部波段聚类。 |
| 聚类数 | 整数 | 否 | `5` | 聚类类别数。 |
| 最大迭代 | 整数 | 否 | `100` | K-Means 最大迭代次数。 |
| 收敛阈值 | 浮点数 | 否 | `0.001` | K-Means 终止阈值。 |

## 算法内容

K-Means 分割通过把每个像元看成一个多维特征点，在特征空间中自动聚成 `k` 类。当前实现不是只用单个波段，而是把输入栅格的全部波段一起作为特征参与聚类，因此输出结果更像“多波段无监督聚类标签图”。

## 处理流程

1. 打开输入栅格并读取全部波段。
2. 将每个像元在所有波段上的值拼成一个特征向量。
3. 把整幅影像展开成“像元数 × 波段数”的样本矩阵。
4. 按 `k`、最大迭代次数和收敛阈值运行 OpenCV K-Means。
5. 得到每个像元所属的聚类标签和聚类紧致度。
6. 将聚类标签重新还原成原始影像尺寸，输出为单波段标签栅格。

## 输出说明

- 输出结果是单波段标签栅格，建议使用 `.tif`。
- 像元值表示聚类标签，通常为 `0` 到 `聚类数-1`。
- 这些标签只代表“分到同一类”，不直接代表具体地物名称。

## 结果解释

- 标签值相同表示这些像元在当前多波段特征空间中被分到了同一类。
- 标签编号本身没有固定语义，不同次运行或不同数据之间不能直接按编号对照地物类别。

## 适用场景

- 无监督预分割、样本探索和多波段影像粗分类。

## 推荐输出文件名

- `kmeans_result.tif`
- `聚类分割结果.tif`

## 注意事项

- `聚类数` 必须大于 0。
- 当前实现不会只取单波段，而是读取全部波段参与聚类。
