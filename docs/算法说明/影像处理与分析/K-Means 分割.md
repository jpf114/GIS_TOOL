# 影像处理与分析 / K-Means 分割

## 功能说明

按所有波段特征对整幅影像做 K-Means 聚类分割。

## 输入与输出

- 输入：多波段或单波段栅格
- 输出：聚类标签栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入影像。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出聚类结果。 |
| 波段号 | 整数 | 否 | `1` | GUI 会显示，但当前实现实际按全部波段聚类。 |
| 聚类数 | 整数 | 否 | `5` | 聚类类别数。 |
| 最大迭代 | 整数 | 否 | `100` | K-Means 最大迭代次数。 |
| 收敛阈值 | 浮点数 | 否 | `0.001` | K-Means 终止阈值。 |

## 输出说明

- 输出结果是单波段标签栅格，建议使用 `.tif`。
- 像元值表示聚类标签，通常为 `0` 到 `聚类数-1`。
- 这些标签只代表“分到同一类”，不直接代表具体地物名称。

## 推荐输出文件名

- `kmeans_result.tif`
- `聚类分割结果.tif`

## 注意事项

- `聚类数` 必须大于 0。
- 当前实现不会只取单波段，而是读取全部波段参与聚类。
