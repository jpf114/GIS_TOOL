# 影像处理与分析 / GLCM 纹理

## 功能说明

按局部窗口计算灰度共生矩阵纹理特征，当前支持 `contrast`、`homogeneity`、`energy`、`entropy`。

## 输入与输出

- 输入：单波段或多波段栅格
- 输出：单波段浮点栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出纹理结果 |
| 波段号 | 整数 | 否 | `1` | 参与处理的波段 |
| 核大小 | 整数 | 否 | `5` | 纹理窗口大小，必须为不小于 3 的奇数 |
| glcm_metric | 枚举 | 否 | `contrast` | 输出纹理指标 |
| glcm_levels | 整数 | 否 | `8` | 量化灰度级数，必须不小于 2 |

## 输出说明

- 输出结果会归一化到 `0` 到 `1`
- 不同 `glcm_metric` 对应不同纹理含义

## 推荐输出文件名

- `glcm_texture_result.tif`

## 注意事项

- `kernel_size` 必须为奇数
- `glcm_levels` 不能小于 2
