# 影像处理与分析 / 影像增强

## 功能说明

执行均衡化、CLAHE、归一化、对数增强、Gamma 校正等增强处理。

## 输入与输出

- 输入：栅格影像
- 输出：增强结果栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入影像。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出结果。 |
| 波段号 | 整数 | 否 | `1` | 从 1 开始。 |
| 增强类型 | 枚举 | 否 | `equalize` | `equalize` `clahe` `normalize` `log` `gamma`。 |
| 裁剪限制 | 浮点数 | 否 | `2.0` | `clahe` 时使用。 |
| Gamma | 浮点数 | 否 | `1.0` | `gamma` 时使用。 |

## 算法内容

影像增强用于改善单波段影像的对比度、亮度分布或细节可见性。当前实现支持全局均衡化、CLAHE、归一化、对数增强和 Gamma 校正几种常见方式，不同方法侧重不同。

## 处理流程

1. 读取输入影像指定波段。
2. 根据所选增强类型进入不同处理分支。
3. `equalize`：转为 8 位灰度后执行直方图均衡化。
4. `clahe`：转为 8 位灰度后执行局部对比度受限均衡化。
5. `normalize`：直接按最小值到最大值归一化到 `0-1`。
6. `log`：先做平移避免负值，再执行对数增强并重新归一化。
7. `gamma`：先归一化，再按 Gamma 指数做亮度校正。
8. 将结果输出为单波段浮点栅格。

## 枚举值说明

### 增强类型

- `equalize`：直方图均衡化
- `clahe`：局部对比度受限均衡化
- `normalize`：归一化
- `log`：对数增强
- `gamma`：Gamma 校正

## 输出说明

- 输出结果为栅格文件。
- 建议优先使用 `.tif` 保存，便于后续在 CLI、GUI 或 GIS 软件中继续处理。

## 结果解释

- 不同增强方式的输出值可用于后续处理，但更主要是改善可视化和局部对比度。
- `gamma` 小于 1 时通常提亮暗部，大于 1 时通常压暗亮部。

## 适用场景

- 单波段显示优化、细节增强和后续分割前预处理。

## 推荐输出文件名

- `影像增强_result.tif`

## 注意事项

- 建议先确认输入波段号、数据类型和 NoData 设置是否正确。
- 输出结果受输入影像质量和参数设置影响较大。
