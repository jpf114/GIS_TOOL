# 影像处理与分析 / 影像增强

## 功能说明

执行均衡化、CLAHE、归一化、对数增强、Gamma 校正等增强处理。

## 输入与输出

- 输入：栅格影像
- 输出：增强结果栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入影像。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出结果。 |
| 波段号 | 整数 | 否 | `1` | 从 1 开始。 |
| 增强类型 | 枚举 | 否 | `equalize` | `equalize` `clahe` `normalize` `log` `gamma`。 |
| 裁剪限制 | 浮点数 | 否 | `2.0` | `clahe` 时使用。 |
| Gamma | 浮点数 | 否 | `1.0` | `gamma` 时使用。 |

## 枚举值说明

### 增强类型

- `equalize`：直方图均衡化
- `clahe`：局部对比度受限均衡化
- `normalize`：归一化
- `log`：对数增强
- `gamma`：Gamma 校正
