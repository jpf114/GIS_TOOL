# 影像处理与分析 / 分水岭分割

## 功能说明

执行分水岭分割，支持外部标记文件，也支持自动生成标记。

## 输入与输出

- 输入：
  - 输入影像
  - 可选标记文件
- 输出：分割标签栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入影像。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出分割结果。 |
| 波段号 | 整数 | 否 | `1` | 从 1 开始。 |
| 标记输入 | 文件路径 | 否 | 无 | 外部分割标记；留空则自动生成。 |

## 输出内容

- 元数据返回 `segment_count`。

## 标记输入要求

如果提供 `标记输入`，应满足：

- 是一幅栅格文件
- 能和输入影像对应到同一空间范围和尺寸
- 其像元值可被转换为整型标记图

常见约定：

- `0`：背景或未标记区域
- `1, 2, 3...`：不同的种子区域编号

简单示意：

```text
0 0 0 0 0
0 1 1 0 2
0 1 1 0 2
0 0 0 0 2
```

上面表示存在两个种子区域，编号分别为 `1` 和 `2`。

如果不提供，代码会自动根据阈值、距离变换和连通域生成标记。

## 输出说明

- 输出结果通常是单波段标签栅格，建议 `.tif`。
- 元数据中的 `segment_count` 表示分割后标签数量。
