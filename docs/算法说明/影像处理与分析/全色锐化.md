# 影像处理与分析 / 全色锐化

## 功能说明

将多光谱影像与全色影像融合，输出更高空间分辨率的多波段结果。

## 输入与输出

- 输入：
  - 输入文件：多光谱影像
  - 全色影像：单波段高分辨率影像
- 输出：融合后的多波段栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 多光谱影像。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出融合结果。 |
| 全色影像 | 文件路径 | 是 | 无 | 全色影像路径。 |
| 融合方法 | 枚举 | 否 | `brovey` | `brovey` `simple_mean` `ihs`。 |

## 注意事项

- 输入多光谱至少要有 2 个波段。
- `ihs` 方法至少要求 3 个波段。
- 若全色与多光谱尺寸不同，代码会把多光谱重采样到全色分辨率。

## 枚举值说明

### 融合方法

- `brovey`：Brovey 融合
- `simple_mean`：简单均值融合
- `ihs`：IHS 融合
