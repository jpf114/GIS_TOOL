# 影像处理与分析 / 全色锐化

## 功能说明

将多光谱影像与全色影像融合，输出更高空间分辨率的多波段结果。

## 输入与输出

- 输入：
  - 输入文件：多光谱影像
  - 全色影像：单波段高分辨率影像
- 输出：融合后的多波段栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 多光谱影像。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出融合结果。 |
| 全色影像 | 文件路径 | 是 | 无 | 全色影像路径。 |
| 融合方法 | 枚举 | 否 | `brovey` | `brovey` `simple_mean` `ihs`。 |

## 算法内容

全色锐化用于把高空间分辨率的全色影像与较低分辨率的多光谱影像融合，得到兼顾空间细节和多波段信息的输出结果。当前实现支持 `brovey`、`simple_mean` 和 `ihs` 三种方式，并在必要时先把多光谱重采样到全色分辨率。

## 处理流程

1. 打开多光谱影像和全色影像。
2. 检查多光谱波段数是否满足当前方法要求。
3. 读取全色波段和多光谱全部波段，并统一转为浮点格式。
4. 如果多光谱尺寸与全色不一致，则先把多光谱重采样到全色分辨率。
5. 按所选方法执行融合：
6. `brovey`：先求多光谱平均强度，再按各波段与平均强度的比值乘上全色。
7. `simple_mean`：各多光谱波段与全色直接做均值融合。
8. `ihs`：将前三个波段转到 HSV 空间，用归一化后的全色替换亮度通道，再转回彩色结果。
9. 将融合后的各波段写出为新的多波段栅格。

## 输出说明

- 输出结果为栅格文件。
- 建议优先使用 `.tif` 保存，便于后续在 CLI、GUI 或 GIS 软件中继续处理。

## 结果解释

- 输出结果空间分辨率与全色影像一致。
- 不同融合方法在细节增强和色彩保持上的表现不同，实际使用时应结合样例对比选择。

## 适用场景

- 遥感影像视觉增强、高分辨率专题判读和融合产品准备。

## 推荐输出文件名

- `全色锐化_result.tif`

## 注意事项

- 输入多光谱至少要有 2 个波段。
- `ihs` 方法至少要求 3 个波段。
- 若全色与多光谱尺寸不同，代码会把多光谱重采样到全色分辨率。

## 枚举值说明

### 融合方法

- `brovey`：Brovey 融合
- `simple_mean`：简单均值融合
- `ihs`：IHS 融合
