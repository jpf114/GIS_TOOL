# 影像处理与分析 / 骨架提取

## 功能说明

对二值目标执行形态学骨架提取，输出表示目标中心线结构的单波段结果。

## 输入与输出
- 输入：栅格影像
- 输出：骨架提取结果栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入栅格影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出骨架提取结果 |
| 波段 | 整数 | 否 | `1` | 参与计算的输入波段 |

## 输出说明

- 输出为单波段骨架结果栅格
- 更适合已经二值化或前景背景差异明显的输入

## 推荐输出文件名
- `skeleton_result.tif`

## 注意事项

- 对灰度影像直接使用时，结果强依赖输入前景质量
- 一般建议先做阈值分割，再做骨架提取
