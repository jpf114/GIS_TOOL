# 影像处理与分析 / 骨架提取

## 功能说明

对二值目标执行形态学骨架提取，输出表示目标中心线结构的单波段结果。

## 输入与输出
- 输入：栅格影像
- 输出：骨架提取结果栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入栅格影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出骨架提取结果 |
| 波段 | 整数 | 否 | `1` | 参与计算的输入波段 |

## 算法内容

骨架提取用于把前景目标逐步细化为近似中心线结构。当前实现会先把输入影像二值化，再通过反复腐蚀、开运算和差分累积的方式提取骨架。

## 处理流程

1. 读取输入影像指定波段。
2. 将波段转为 8 位灰度图。
3. 对灰度图执行 Otsu 二值化，得到前景掩膜。
4. 初始化空骨架图和工作图。
5. 反复对工作图做腐蚀。
6. 对腐蚀结果做膨胀，再与原工作图做差，提取本轮被剥离出的骨架部分。
7. 将本轮骨架部分累积到总骨架图中。
8. 当工作图全部腐蚀为空时结束循环。
9. 将最终骨架结果输出为单波段栅格。

## 输出说明

- 输出为单波段骨架结果栅格
- 更适合已经二值化或前景背景差异明显的输入

## 结果解释

- 输出保留的是前景目标的细化中心线结构。
- 对噪声多或前景粘连严重的输入，骨架结果会明显受影响。

## 适用场景

- 中心线提取、细长结构分析和后续拓扑检查前处理。

## 推荐输出文件名
- `skeleton_result.tif`

## 注意事项

- 对灰度影像直接使用时，结果强依赖输入前景质量
- 一般建议先做阈值分割，再做骨架提取
