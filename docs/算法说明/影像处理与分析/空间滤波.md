# 影像处理与分析 / 空间滤波

## 功能说明

对影像执行平滑、形态学等空间滤波。

## 输入与输出

- 输入：栅格影像
- 输出：滤波结果栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入影像。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出结果。 |
| 波段号 | 整数 | 否 | `1` | 从 1 开始。 |
| 滤波类型 | 枚举 | 否 | `gaussian` | `gaussian` `median` `bilateral` `morph_open` `morph_close` `morph_dilate` `morph_erode`。 |
| 核大小 | 整数 | 否 | `5` | 滤波核大小。 |
| Sigma | 浮点数 | 否 | `1.5` | 高斯等方法使用。 |

## 算法内容

空间滤波用于在局部邻域上平滑噪声、保留边缘或做形态学结构处理。当前实现支持高斯、中值、双边以及开闭运算、膨胀、腐蚀等多种方式，不同滤波器适合不同目标。

## 处理流程

1. 读取输入影像指定波段。
2. 对核大小做最小值和奇数修正，保证适合后续滤波。
3. 根据滤波类型进入不同处理分支。
4. `gaussian`：直接对浮点影像执行高斯平滑。
5. `median`、`bilateral`：先转为 8 位灰度，再执行滤波，最后转回浮点。
6. 形态学开闭、膨胀、腐蚀：先转为 8 位灰度，构造矩形核后执行形态学处理，再转回浮点。
7. 将处理结果输出为单波段栅格。

## 输出说明

- 输出结果为栅格文件。
- 建议优先使用 `.tif` 保存，便于后续在 CLI、GUI 或 GIS 软件中继续处理。

## 结果解释

- 平滑型滤波更适合去噪。
- 形态学滤波更适合处理目标形状、孔洞和局部连接关系。

## 适用场景

- 去噪、边界清理、形态优化和后续分割前处理。

## 推荐输出文件名

- `空间滤波_result.tif`

## 注意事项

- 结果按单波段栅格写出。

## 枚举值说明

### 滤波类型

- `gaussian`：高斯滤波
- `median`：中值滤波
- `bilateral`：双边滤波
- `morph_open`：开运算
- `morph_close`：闭运算
- `morph_dilate`：膨胀
- `morph_erode`：腐蚀
