# 影像处理与分析 / 空间滤波

## 功能说明

对影像执行平滑、形态学等空间滤波。

## 输入与输出

- 输入：栅格影像
- 输出：滤波结果栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入影像。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出结果。 |
| 波段号 | 整数 | 否 | `1` | 从 1 开始。 |
| 滤波类型 | 枚举 | 否 | `gaussian` | `gaussian` `median` `bilateral` `morph_open` `morph_close` `morph_dilate` `morph_erode`。 |
| 核大小 | 整数 | 否 | `5` | 滤波核大小。 |
| Sigma | 浮点数 | 否 | `1.5` | 高斯等方法使用。 |

## 输出说明

- 输出结果为栅格文件。
- 建议优先使用 `.tif` 保存，便于后续在 CLI、GUI 或 GIS 软件中继续处理。

## 推荐输出文件名

- `空间滤波_result.tif`

## 注意事项

- 结果按单波段栅格写出。

## 枚举值说明

### 滤波类型

- `gaussian`：高斯滤波
- `median`：中值滤波
- `bilateral`：双边滤波
- `morph_open`：开运算
- `morph_close`：闭运算
- `morph_dilate`：膨胀
- `morph_erode`：腐蚀
