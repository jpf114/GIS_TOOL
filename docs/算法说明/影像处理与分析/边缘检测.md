# 影像处理与分析 / 边缘检测

## 功能说明

执行 Canny、Sobel、Laplacian、Scharr 边缘检测。

## 输入与输出

- 输入：栅格影像
- 输出：边缘结果栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入影像。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出结果。 |
| 波段号 | 整数 | 否 | `1` | 从 1 开始。 |
| 边缘方法 | 枚举 | 否 | `canny` | `canny` `sobel` `laplacian` `scharr`。 |
| 低阈值 | 浮点数 | 否 | `50.0` | `canny` 常用。 |
| 高阈值 | 浮点数 | 否 | `150.0` | `canny` 常用。 |
| Sobel dx | 整数 | 否 | `1` | `sobel`/`scharr` 使用。 |
| Sobel dy | 整数 | 否 | `1` | `sobel`/`scharr` 使用。 |

## 算法内容

边缘检测用于突出灰度变化明显的位置。当前实现支持 Canny、Sobel、Laplacian 和 Scharr 四种方法，输出的是单波段边缘响应结果图。

## 处理流程

1. 读取输入影像指定波段。
2. 将波段转为 8 位灰度图。
3. 根据所选边缘方法进入不同处理分支。
4. `canny`：按给定高低阈值执行 Canny 检测。
5. `sobel`：按 `dx/dy` 方向计算梯度，再转为可显示结果。
6. `laplacian`：计算二阶导数边缘。
7. `scharr`：按更强的梯度算子计算方向边缘。
8. 将结果转为 `0-1` 左右的浮点输出栅格。

## 枚举值说明

### 边缘方法

- `canny`：Canny 边缘检测
- `sobel`：Sobel 边缘
- `laplacian`：Laplacian 边缘
- `scharr`：Scharr 边缘

## 输出说明

- 输出结果为栅格文件。
- 建议优先使用 `.tif` 保存，便于后续在 CLI、GUI 或 GIS 软件中继续处理。

## 结果解释

- 输出值越高，通常表示该位置边缘响应越强。
- 不同方法对噪声、细线和强边界的敏感性不同。

## 适用场景

- 边界提取、目标轮廓增强和后续线性结构分析前处理。

## 推荐输出文件名

- `边缘检测_result.tif`

## 注意事项

- 建议先确认输入波段号、数据类型和 NoData 设置是否正确。
- 输出结果受输入影像质量和参数设置影响较大。
