# 影像处理与分析 / 阈值分割

## 功能说明

对指定波段执行阈值分割，输出分割结果栅格。

## 输入与输出

- 输入：栅格影像
- 输出：单波段结果栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入影像。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出结果。 |
| 波段号 | 整数 | 否 | `1` | 从 1 开始。 |
| 方法 | 枚举 | 否 | 见说明 | `binary` `binary_inv` `truncate` `tozero` `otsu` `adaptive_gaussian` `adaptive_mean`。 |
| 阈值值 | 浮点数 | 否 | `128.0` | 普通阈值法时使用。 |
| 最大值 | 浮点数 | 否 | `255.0` | 阈值后赋值上限。 |

## 输出说明

- 输出结果为栅格文件。
- 建议优先使用 `.tif` 保存，便于后续在 CLI、GUI 或 GIS 软件中继续处理。

## 推荐输出文件名

- `阈值分割_result.tif`

## 注意事项

- `otsu` 与自适应阈值会忽略固定阈值值。
- 这里有一处实现细节需要注意：
  - 执行层在外部完全不传 `method` 时，会回退到 `otsu`
  - 但 GUI 组合框当前会默认落到第一项，因此界面实际更接近默认 `binary`

## 枚举值说明

### 方法

- `binary`：大于阈值置为最大值
- `binary_inv`：反向二值化
- `truncate`：高于阈值截断到阈值
- `tozero`：低于阈值置零
- `otsu`：自动阈值
- `adaptive_gaussian`：自适应高斯阈值
- `adaptive_mean`：自适应均值阈值
