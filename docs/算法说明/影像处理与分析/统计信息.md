# 影像处理与分析 / 统计信息

## 功能说明

统计指定波段的基础数值信息，并在结果面板显示。

## 输入与输出

- 输入：栅格影像
- 输出：无文件输出，仅返回文本和元数据

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入影像。 |
| 波段号 | 整数 | 否 | `1` | 从 1 开始。 |

## 算法内容

统计信息不是改写型算法，而是对指定波段执行一次基础数值检查。当前实现会直接读取波段统计结果、数据类型、尺寸和 NoData 信息，并以文本和元数据形式返回。

## 处理流程

1. 打开输入栅格并定位指定波段。
2. 调用波段统计流程，得到最小值、最大值、均值和标准差。
3. 读取栅格宽高、波段数据类型和 NoData 值。
4. 将这些信息整理成文本摘要和元数据返回，不生成输出文件。

## 返回内容

- 最小值、最大值、均值、标准差
- 波段类型
- 栅格宽高
- NoData 值（若存在）

## 返回字段说明

- `min`：最小值
- `max`：最大值
- `mean`：均值
- `stddev`：标准差
- `width`：栅格宽度
- `height`：栅格高度
- `type`：波段数据类型，如 `Byte (uint8)`、`Float32`
- `nodata`：NoData 值，仅在原始波段存在 NoData 时返回

## 输出说明

- 本功能不生成输出文件。
- 统计结果直接显示在执行结果中，并附带对应元数据字段。

## 结果解释

- 结果适合快速判断波段值域、数据类型和是否存在异常分布。
- 如果存在 NoData，返回结果会显式带出 NoData 值。

## 适用场景

- 数据检查、阈值选择和处理前快速质检。

## 注意事项

- 当前只统计一个波段，默认统计第 `1` 波段。
- 本功能没有输出文件参数，结果只在执行结果面板中显示。
