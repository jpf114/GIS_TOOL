# 影像处理与分析 / 霍夫变换

## 功能说明

检测输入影像中的直线或圆形结构，并输出检测结果掩膜。

## 输入与输出

- 输入：栅格影像
- 输出：检测结果栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入影像。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出检测结果。 |
| 波段号 | 整数 | 否 | `1` | 从 1 开始。 |
| 霍夫类型 | 枚举 | 否 | `lines` | `lines` 或 `circles`。 |
| 霍夫阈值 | 浮点数 | 否 | `50.0` | 累加器阈值。 |
| 最小线长 | 浮点数 | 否 | `50.0` | 直线检测使用。 |
| 最大线间隙 | 浮点数 | 否 | `10.0` | 直线检测使用。 |
| 最小半径 | 整数 | 否 | `1` | 圆检测使用。 |
| 最大半径 | 整数 | 否 | `0` | 0 近似表示不额外限制。 |
| 圆检测阈值 | 浮点数 | 否 | `30.0` | 圆检测累加器阈值。 |

## 输出内容

- 元数据返回检测类型和检测数量。

## 枚举值说明

### 霍夫类型

- `lines`：检测直线
- `circles`：检测圆
