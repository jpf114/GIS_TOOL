# 影像处理与分析 / 波段运算

## 功能说明

按表达式对多波段栅格执行像元级计算。

## 输入与输出

- 输入：多波段栅格
- 输出：运算结果栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入影像。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出结果。 |
| 表达式 | 字符串 | 是 | 无 | 例如 `B1+B2`、`B1*0.5+B2*0.5`。 |

## 注意事项

- 表达式中的波段占位符使用 `B1`、`B2` 这种形式。
