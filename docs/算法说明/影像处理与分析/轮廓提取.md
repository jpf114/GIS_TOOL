# 影像处理与分析 / 轮廓提取

## 功能说明

先对输入影像做 Otsu 二值化，再按最小面积筛选轮廓并输出结果栅格。

## 输入与输出

- 输入：栅格影像
- 输出：轮廓掩膜栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入影像。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出结果。 |
| 波段号 | 整数 | 否 | `1` | 从 1 开始。 |
| 最小面积 | 浮点数 | 否 | `100.0` | 小于该面积的轮廓会被丢弃。 |

## 算法内容

轮廓提取用于从灰度影像中提取独立目标的外部区域轮廓。当前实现先对影像做 Otsu 二值化，再找出外部轮廓，并按最小面积筛掉太小的目标，最终输出一个填充后的轮廓掩膜图。

## 处理流程

1. 读取输入影像指定波段。
2. 将波段转为 8 位灰度图。
3. 对灰度图执行 Otsu 二值化，得到前景掩膜。
4. 在二值图上提取外部轮廓。
5. 逐个计算轮廓面积。
6. 只保留面积大于等于 `最小面积` 的轮廓，并将其填充绘制到结果图中。
7. 将结果归一化到浮点输出，并记录总轮廓数和保留轮廓数。

## 输出内容

- 元数据返回总轮廓数与保留轮廓数。

## 输出说明

- 输出结果为栅格文件。
- 建议优先使用 `.tif` 保存，便于后续在 CLI、GUI 或 GIS 软件中继续处理。

## 结果解释

- 输出是一个掩膜型结果图，不是矢量轮廓文件。
- 值为前景的位置表示被保留的轮廓区域，其他位置为背景。

## 适用场景

- 独立目标区域提取、面积筛选和目标前景粗分割。

## 推荐输出文件名

- `轮廓提取_result.tif`

## 注意事项

- 建议先确认输入波段号、数据类型和 NoData 设置是否正确。
- 输出结果受输入影像质量和参数设置影响较大。
