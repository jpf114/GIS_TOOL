# 影像处理与分析 / 轮廓提取

## 功能说明

先对输入影像做 Otsu 二值化，再按最小面积筛选轮廓并输出结果栅格。

## 输入与输出

- 输入：栅格影像
- 输出：轮廓掩膜栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入影像。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出结果。 |
| 波段号 | 整数 | 否 | `1` | 从 1 开始。 |
| 最小面积 | 浮点数 | 否 | `100.0` | 小于该面积的轮廓会被丢弃。 |

## 输出内容

- 元数据返回总轮廓数与保留轮廓数。

## 输出说明

- 输出结果为栅格文件。
- 建议优先使用 `.tif` 保存，便于后续在 CLI、GUI 或 GIS 软件中继续处理。

## 推荐输出文件名

- `轮廓提取_result.tif`

## 注意事项

- 建议先确认输入波段号、数据类型和 NoData 设置是否正确。
- 输出结果受输入影像质量和参数设置影响较大。
