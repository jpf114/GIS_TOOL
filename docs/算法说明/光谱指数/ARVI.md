# 光谱指数 / ARVI

## 功能说明

根据蓝光、红光和近红外波段计算 ARVI，适合在大气影响较明显时做植被指数估计。

## 输入与输出
- 输入：多波段栅格
- 输出：ARVI 栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入多波段影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出 ARVI 结果 |
| 蓝光波段 | 整数 | 否 | `1` | 蓝光波段序号 |
| 红光波段 | 整数 | 否 | `3` | 红光波段序号 |
| 近红外波段 | 整数 | 否 | `4` | 近红外波段序号 |

## 算法内容

ARVI 通过引入蓝光参与修正，减弱部分大气影响对植被指数的干扰。当前实现按标准 ARVI 公式逐像元计算，输出单波段浮点指数图。

## 处理流程

1. 打开输入栅格。
2. 检查蓝光、红光和近红外波段号是否有效。
3. 读取蓝光、红光和近红外波段数据。
4. 先计算红蓝修正项 `(2 * Red - Blue)`。
5. 对每个像元按 `(NIR - (2 * Red - Blue)) / (NIR + (2 * Red - Blue))` 计算 ARVI。
6. 对分母接近零的位置按安全除法逻辑处理。
7. 输出为单波段浮点栅格。

## 输出说明

- 计算公式：`(NIR - (2 * Red - Blue)) / (NIR + (2 * Red - Blue))`
- 输出元数据会返回指数最小值、最大值

## 结果解释

- 较高值通常表示植被活性更强。
- 由于结果依赖蓝光质量，若蓝光噪声较大或大气条件复杂，实际效果也会受影响。

## 适用场景

- 需要减弱部分大气影响的植被分析场景。

## 推荐输出文件名
- `arvi_result.tif`

## 注意事项

- 蓝光波段质量较差时，ARVI 的稳定性会受到影响
