# 光谱指数 / NDBI

## 功能说明

根据短波红外 1 与近红外波段计算 NDBI，适合建设用地和不透水面提取。

## 输入与输出
- 输入：多波段栅格
- 输出：NDBI 栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入多波段影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出 NDBI 结果 |
| 近红外波段 | 整数 | 否 | `4` | 近红外波段序号 |
| 短波红外1波段 | 整数 | 否 | `5` | SWIR1 波段序号 |

## 算法内容

NDBI 通过比较短波红外 1 与近红外的差异，增强建设用地和不透水面的表现。当前实现按标准公式逐像元计算，输出单波段浮点指数图。

## 处理流程

1. 打开输入多波段栅格。
2. 检查近红外和短波红外 1 波段号是否有效。
3. 读取近红外与短波红外 1 波段数据。
4. 对每个像元按 `(SWIR1 - NIR) / (SWIR1 + NIR)` 计算 NDBI。
5. 对分母接近零的位置按安全除法逻辑处理。
6. 输出为单波段浮点栅格。

## 输出说明

- 计算公式：`(SWIR1 - NIR) / (SWIR1 + NIR)`
- 输出元数据会返回指数最小值、最大值

## 结果解释

- 较高值通常更容易对应建设用地、不透水面或部分裸露地表。
- 裸地与建筑在某些区域可能同时表现较高，因此建议与 NDVI、NDWI 等一起判读。

## 适用场景

- 城市扩展、不透水面识别和建设区辅助提取。

## 推荐输出文件名
- `ndbi_result.tif`

## 注意事项

- 在部分数据上，NDBI 需要结合 NDVI 或水体指数一起判读，避免建筑与裸地混淆
