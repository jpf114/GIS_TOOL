# 光谱指数 / NDBI

## 功能说明

根据短波红外 1 与近红外波段计算 NDBI，适合建设用地和不透水面提取。

## 输入与输出
- 输入：多波段栅格
- 输出：NDBI 栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入多波段影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出 NDBI 结果 |
| 近红外波段 | 整数 | 否 | `4` | 近红外波段序号 |
| 短波红外1波段 | 整数 | 否 | `5` | SWIR1 波段序号 |

## 输出说明

- 计算公式：`(SWIR1 - NIR) / (SWIR1 + NIR)`
- 输出元数据会返回指数最小值、最大值

## 推荐输出文件名
- `ndbi_result.tif`

## 注意事项

- 在部分数据上，NDBI 需要结合 NDVI 或水体指数一起判读，避免建筑与裸地混淆
