# 光谱指数 / MNDWI

## 功能说明

根据绿光与短波红外 1 波段计算 MNDWI，适合增强水体与建筑物、裸地之间的区分度。

## 输入与输出
- 输入：多波段栅格
- 输出：MNDWI 栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入多波段影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出 MNDWI 结果 |
| 绿光波段 | 整数 | 否 | `2` | 绿光波段序号 |
| 短波红外1波段 | 整数 | 否 | `5` | SWIR1 波段序号 |

## 算法内容

MNDWI 用短波红外 1 替代 NDWI 中的近红外，目的是在一些场景下更好地区分水体与建筑、裸地等非水目标。当前实现按标准公式逐像元计算，输出单波段浮点指数图。

## 处理流程

1. 打开输入多波段栅格。
2. 检查绿光和短波红外 1 波段号是否有效。
3. 读取绿光与短波红外 1 波段数据。
4. 对每个像元按 `(Green - SWIR1) / (Green + SWIR1)` 计算 MNDWI。
5. 对分母接近零的位置按安全除法逻辑处理。
6. 将结果写出为单波段浮点栅格。

## 输出说明

- 计算公式：`(Green - SWIR1) / (Green + SWIR1)`
- 输出元数据会返回指数最小值、最大值

## 结果解释

- 水体在很多场景下会表现为更高值。
- 与 NDWI 相比，它通常对建筑和裸地抑制更强，但仍需结合具体数据验证。

## 适用场景

- 城市环境水体提取、河湖识别和湿地区域分析。

## 推荐输出文件名
- `mndwi_result.tif`

## 注意事项

- 如果输入影像没有 SWIR1 波段，则不能直接使用该指数
