# 光谱指数 / 自定义指数

## 功能说明

根据用户提供的表达式或内置预设计算自定义光谱指数，适合快速试验业务口径或临时组合波段。

## 输入与输出
- 输入：多波段栅格
- 输出：自定义指数栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入多波段影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出自定义指数结果 |
| 表达式 | 字符串 | 否 | 空 | 自定义表达式，例如 `(B4-B3)/(B4+B3)` |
| 预设表达式 | 枚举 | 否 | `none` | 内置表达式预设 |
| 蓝光波段 | 整数 | 否 | `1` | `BLUE` 别名对应波段 |
| 绿光波段 | 整数 | 否 | `2` | `GREEN` 别名对应波段 |
| 红光波段 | 整数 | 否 | `3` | `RED` 别名对应波段 |
| 近红外波段 | 整数 | 否 | `4` | `NIR` 别名对应波段 |
| 短波红外1波段 | 整数 | 否 | `5` | `SWIR1` 别名对应波段 |

## 输出说明

- 支持直接使用波段编号别名：`B1`、`B2`、`B3`
- 支持使用语义别名：`BLUE`、`GREEN`、`RED`、`NIR`、`SWIR1`
- 内置预设包括：
  - `ndvi_alias`
  - `ndmi_alias`
  - `ndwi_alias`
  - `mndwi_alias`
  - `ndbi_alias`
  - `bsi_alias`
  - `gndvi_alias`
  - `savi_alias`
  - `evi_alias`
  - `evi2_alias`

## 推荐输出文件名
- `custom_index_result.tif`

## 注意事项

- `expression` 为空时会自动使用 `preset`
- 如果表达式引用了 `BLUE`、`RED` 这类别名，需要同时保证对应波段号设置正确
- 当前表达式支持基础四则运算和括号
