# 光谱指数 / NBR

## 功能说明

根据近红外与短波红外 2 波段计算 NBR，适合火烧迹地和植被受损区域识别。

## 输入与输出
- 输入：多波段栅格
- 输出：NBR 栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入多波段影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出 NBR 结果 |
| 近红外波段 | 整数 | 否 | `4` | 近红外波段序号 |
| 短波红外2波段 | 整数 | 否 | `6` | SWIR2 波段序号 |

## 算法内容

NBR 通过比较近红外与短波红外 2 的差异，常用于识别火烧迹地和植被受损区域。当前实现按标准公式逐像元计算，输出单波段浮点指数图。

## 处理流程

1. 打开输入多波段栅格。
2. 检查近红外和短波红外 2 波段号是否有效。
3. 读取近红外与短波红外 2 波段数据。
4. 对每个像元按 `(NIR - SWIR2) / (NIR + SWIR2)` 计算 NBR。
5. 对分母接近零的位置按安全除法逻辑处理。
6. 输出为单波段浮点栅格。

## 输出说明

- 计算公式：`(NIR - SWIR2) / (NIR + SWIR2)`
- 输出元数据会返回指数最小值、最大值

## 结果解释

- 较高值通常更接近健康植被响应。
- 受烧毁、受损或干扰明显的区域，往往会表现为较低值。

## 适用场景

- 火情后植被受损分析和烧毁区域判读。

## 推荐输出文件名
- `nbr_result.tif`

## 注意事项

- 如果影像没有 SWIR2 波段，不能直接使用该指数
