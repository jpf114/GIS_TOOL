# 光谱指数 / UI

## 功能说明

根据短波红外 2 与近红外波段计算 UI，适合城镇不透水面识别的辅助分析。

## 输入与输出
- 输入：多波段栅格
- 输出：UI 栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入多波段影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出 UI 结果 |
| 近红外波段 | 整数 | 否 | `4` | 近红外波段序号 |
| 短波红外2波段 | 整数 | 否 | `6` | SWIR2 波段序号 |

## 算法内容

UI 用短波红外 2 与近红外的差异来辅助增强城市建设区和不透水面响应。当前实现按标准公式逐像元计算，输出单波段浮点指数图。

## 处理流程

1. 打开输入多波段栅格。
2. 检查近红外和短波红外 2 波段号是否有效。
3. 读取近红外与短波红外 2 波段数据。
4. 对每个像元按 `(SWIR2 - NIR) / (SWIR2 + NIR)` 计算 UI。
5. 对分母接近零的位置按安全除法逻辑处理。
6. 输出为单波段浮点栅格。

## 输出说明

- 计算公式：`(SWIR2 - NIR) / (SWIR2 + NIR)`
- 输出元数据会返回指数最小值、最大值

## 结果解释

- 较高值通常更容易对应城市建成区或不透水面。
- 单独使用时容易与部分裸地混淆，更适合作为辅助指数。

## 适用场景

- 城市建设区、不透水面辅助判读和专题组合分析。

## 推荐输出文件名
- `ui_result.tif`

## 注意事项

- UI 更适合作为辅助指数，与 NDBI、NDVI 联合使用效果更稳定
