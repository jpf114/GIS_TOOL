# 光谱指数 / UI

## 功能说明

根据短波红外 2 与近红外波段计算 UI，适合城镇不透水面识别的辅助分析。

## 输入与输出
- 输入：多波段栅格
- 输出：UI 栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入多波段影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出 UI 结果 |
| 近红外波段 | 整数 | 否 | `4` | 近红外波段序号 |
| 短波红外2波段 | 整数 | 否 | `6` | SWIR2 波段序号 |

## 输出说明

- 计算公式：`(SWIR2 - NIR) / (SWIR2 + NIR)`
- 输出元数据会返回指数最小值、最大值

## 推荐输出文件名
- `ui_result.tif`

## 注意事项

- UI 更适合作为辅助指数，与 NDBI、NDVI 联合使用效果更稳定
