# 光谱指数 / BI

## 功能说明

根据红光与近红外波段计算 BI，适合裸地和高反射地表的辅助识别。

## 输入与输出
- 输入：多波段栅格
- 输出：BI 栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入多波段影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出 BI 结果 |
| 红光波段 | 整数 | 否 | `3` | 红光波段序号 |
| 近红外波段 | 整数 | 否 | `4` | 近红外波段序号 |

## 算法内容

BI 用红光和近红外构造一个与特定参考点距离相关的指数，用于辅助识别裸地和高反射地表。当前实现按固定公式逐像元计算，输出单波段浮点结果图。由于它不是归一化差值形式，值域可能明显大于 `1`。

## 处理流程

1. 打开输入多波段栅格。
2. 检查红光和近红外波段号是否有效。
3. 读取红光与近红外波段数据。
4. 先分别计算 `(0.1 - Red)` 和 `(0.06 - NIR)`。
5. 对每个像元按 `1 / ((0.1 - Red)^2 + (0.06 - NIR)^2)` 计算 BI。
6. 为避免分母过小带来异常值，计算时会加入很小的稳定项。
7. 输出为单波段浮点栅格。

## 输出说明

- 计算公式：`1 / ((0.1 - Red)^2 + (0.06 - NIR)^2)`
- 输出元数据会返回指数最小值、最大值

## 结果解释

- 值越高，表示该像元在当前公式设定下越接近目标反射特征。
- 由于值域不固定，实际使用时更适合看相对强弱或结合阈值试验。

## 适用场景

- 裸地、高反射区域或特定地表模式的辅助识别。

## 推荐输出文件名
- `bi_result.tif`

## 注意事项

- BI 不是标准归一化指数，结果值域可能明显大于 `1`
