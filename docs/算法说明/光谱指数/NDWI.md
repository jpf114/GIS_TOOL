# 光谱指数 / NDWI

## 功能说明

根据绿光与近红外波段计算 NDWI，适合水体提取和湿润区域识别。

## 输入与输出
- 输入：多波段栅格
- 输出：NDWI 栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入多波段影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出 NDWI 结果 |
| 绿光波段 | 整数 | 否 | `2` | 绿光波段序号 |
| 近红外波段 | 整数 | 否 | `4` | 近红外波段序号 |

## 算法内容

当前实现采用 McFeeters 口径的 NDWI，通过比较绿光与近红外的差异来增强水体响应。算法按标准公式逐像元计算，输出的是单波段浮点指数图，不包含阈值提水或后分类步骤。

## 处理流程

1. 打开输入多波段栅格。
2. 检查绿光和近红外波段号是否有效。
3. 读取绿光与近红外波段数据。
4. 对每个像元按 `(Green - NIR) / (Green + NIR)` 计算 NDWI。
5. 对分母接近零的位置按安全除法逻辑处理。
6. 输出为单波段浮点栅格。

## 输出说明

- 计算公式：`(Green - NIR) / (Green + NIR)`
- 输出元数据会返回指数最小值、最大值

## 结果解释

- 水体区域通常更容易表现为较高值。
- 建筑、裸地和植被在不同场景下可能与浅水或阴影发生混淆，实际判读建议结合其他指数。

## 适用场景

- 水体快速提取、湿润区域识别和基础水系判读。

## 推荐输出文件名
- `ndwi_result.tif`

## 注意事项

- 该实现采用 McFeeters 口径，使用绿光和近红外波段
