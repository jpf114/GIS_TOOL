# 光谱指数 / NDWI

## 功能说明

根据绿光与近红外波段计算 NDWI，适合水体提取和湿润区域识别。

## 输入与输出
- 输入：多波段栅格
- 输出：NDWI 栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入多波段影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出 NDWI 结果 |
| 绿光波段 | 整数 | 否 | `2` | 绿光波段序号 |
| 近红外波段 | 整数 | 否 | `4` | 近红外波段序号 |

## 输出说明

- 计算公式：`(Green - NIR) / (Green + NIR)`
- 输出元数据会返回指数最小值、最大值

## 推荐输出文件名
- `ndwi_result.tif`

## 注意事项

- 该实现采用 McFeeters 口径，使用绿光和近红外波段
