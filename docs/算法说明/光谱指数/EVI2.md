# 光谱指数 / EVI2

## 功能说明

根据红光与近红外波段计算双波段增强植被指数 EVI2，适合缺少蓝光波段的多光谱数据。

## 输入与输出

- 输入：多波段栅格
- 输出：EVI2 栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入多波段影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出 EVI2 结果 |
| 红光波段 | 整数 | 否 | `3` | 红光波段序号 |
| 近红外波段 | 整数 | 否 | `4` | 近红外波段序号 |

## 算法内容

EVI2 是 EVI 的双波段近似形式，在缺少蓝光波段时仍可实现增强植被指数计算。当前实现按固定 EVI2 公式逐像元计算，输出单波段浮点指数图。

## 处理流程

1. 打开输入多波段栅格。
2. 检查红光和近红外波段号是否有效。
3. 读取红光与近红外波段数据。
4. 对每个像元按 `2.5 * (NIR - Red) / (NIR + 2.4 * Red + 1)` 计算 EVI2。
5. 对分母接近零的位置按安全除法逻辑处理。
6. 输出为单波段浮点栅格。

## 输出说明

- 计算公式：`2.5 * (NIR - Red) / (NIR + 2.4 * Red + 1)`
- 常用于快速替代 EVI

## 结果解释

- 较高值通常表示植被活性更强。
- 当数据缺少蓝光波段时，EVI2 是比 NDVI 更接近 EVI 思路的替代选择。

## 适用场景

- 缺少蓝光波段的多光谱数据植被分析。

## 推荐输出文件名

- `evi2_result.tif`

## 注意事项

- 红光和近红外波段号必须落在实际波段范围内
- 结果元数据会返回最小值、最大值以及实际使用的波段号
