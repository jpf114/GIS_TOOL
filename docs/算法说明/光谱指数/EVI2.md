# 光谱指数 / EVI2

## 功能说明

根据红光与近红外波段计算双波段增强植被指数 EVI2，适合缺少蓝光波段的多光谱数据。

## 输入与输出

- 输入：多波段栅格
- 输出：EVI2 栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入多波段影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出 EVI2 结果 |
| 红光波段 | 整数 | 否 | `3` | 红光波段序号 |
| 近红外波段 | 整数 | 否 | `4` | 近红外波段序号 |

## 输出说明

- 计算公式：`2.5 * (NIR - Red) / (NIR + 2.4 * Red + 1)`
- 常用于快速替代 EVI

## 推荐输出文件名

- `evi2_result.tif`

## 注意事项

- 红光和近红外波段号必须落在实际波段范围内
- 结果元数据会返回最小值、最大值以及实际使用的波段号
