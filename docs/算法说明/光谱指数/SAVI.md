# 光谱指数 / SAVI

## 功能说明

根据红光与近红外波段计算 SAVI，适合在植被覆盖较稀疏、土壤背景影响较明显时使用。

## 输入与输出
- 输入：多波段栅格
- 输出：SAVI 栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入多波段影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出 SAVI 结果 |
| 红光波段 | 整数 | 否 | `3` | 红光波段序号 |
| 近红外波段 | 整数 | 否 | `4` | 近红外波段序号 |
| L 参数 | 浮点数 | 否 | `0.5` | 土壤亮度修正项 |

## 输出说明

- 计算公式：`((NIR - Red) / (NIR + Red + L)) * (1 + L)`
- 输出元数据会返回指数最小值、最大值

## 推荐输出文件名
- `savi_result.tif`

## 注意事项

- `L` 越大，土壤背景影响抑制越强
- 在高覆盖植被场景中，SAVI 与 NDVI 结果通常接近
