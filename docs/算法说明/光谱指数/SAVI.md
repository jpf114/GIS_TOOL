# 光谱指数 / SAVI

## 功能说明

根据红光与近红外波段计算 SAVI，适合在植被覆盖较稀疏、土壤背景影响较明显时使用。

## 输入与输出
- 输入：多波段栅格
- 输出：SAVI 栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入多波段影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出 SAVI 结果 |
| 红光波段 | 整数 | 否 | `3` | 红光波段序号 |
| 近红外波段 | 整数 | 否 | `4` | 近红外波段序号 |
| L 参数 | 浮点数 | 否 | `0.5` | 土壤亮度修正项 |

## 算法内容

SAVI 在 NDVI 基础上引入土壤背景修正项 `L`，用于减弱植被覆盖较低时土壤亮度对结果的干扰。当前实现按标准 SAVI 公式逐像元计算，输出单波段浮点指数图。

## 处理流程

1. 打开输入多波段栅格。
2. 检查红光和近红外波段号是否有效。
3. 读取红光与近红外波段数据。
4. 根据当前 `L` 参数，对每个像元按 `((NIR - Red) / (NIR + Red + L)) * (1 + L)` 计算 SAVI。
5. 对分母接近零的位置按安全除法逻辑处理。
6. 输出为单波段浮点栅格。

## 输出说明

- 计算公式：`((NIR - Red) / (NIR + Red + L)) * (1 + L)`
- 输出元数据会返回指数最小值、最大值

## 结果解释

- 较高值通常表示植被更明显。
- `L` 越大，对土壤背景的抑制越强，但也会改变结果幅度。

## 适用场景

- 植被覆盖较稀疏、土壤背景影响明显的分析场景。

## 推荐输出文件名
- `savi_result.tif`

## 注意事项

- `L` 越大，土壤背景影响抑制越强
- 在高覆盖植被场景中，SAVI 与 NDVI 结果通常接近
