# 光谱指数 / BSI

## 功能说明

根据蓝光、红光、近红外与短波红外 1 波段计算裸土指数 BSI，适合辅助识别裸地和低覆盖区域。

## 输入与输出

- 输入：多波段栅格
- 输出：BSI 栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入多波段影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出 BSI 结果 |
| 蓝光波段 | 整数 | 否 | `1` | 蓝光波段序号 |
| 红光波段 | 整数 | 否 | `3` | 红光波段序号 |
| 近红外波段 | 整数 | 否 | `4` | 近红外波段序号 |
| 短波红外1波段 | 整数 | 否 | `5` | 短波红外1波段序号 |

## 输出说明

- 计算公式：`((SWIR1 + Red) - (NIR + Blue)) / ((SWIR1 + Red) + (NIR + Blue))`
- 一般可与 NDVI、NDBI 联合判读

## 推荐输出文件名

- `bsi_result.tif`

## 注意事项

- 相关波段号必须落在实际波段范围内
- 结果元数据会返回最小值、最大值以及实际使用的波段号
