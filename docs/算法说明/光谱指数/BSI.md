# 光谱指数 / BSI

## 功能说明

根据蓝光、红光、近红外与短波红外 1 波段计算裸土指数 BSI，适合辅助识别裸地和低覆盖区域。

## 输入与输出

- 输入：多波段栅格
- 输出：BSI 栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入多波段影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出 BSI 结果 |
| 蓝光波段 | 整数 | 否 | `1` | 蓝光波段序号 |
| 红光波段 | 整数 | 否 | `3` | 红光波段序号 |
| 近红外波段 | 整数 | 否 | `4` | 近红外波段序号 |
| 短波红外1波段 | 整数 | 否 | `5` | 短波红外1波段序号 |

## 算法内容

BSI 通过联合蓝光、红光、近红外和短波红外 1，增强裸地和低覆盖区域的响应。当前实现按经典 BSI 公式逐像元计算，输出单波段浮点指数图。

## 处理流程

1. 打开输入多波段栅格。
2. 检查蓝光、红光、近红外和短波红外 1 波段号是否有效。
3. 读取四个参与计算的波段。
4. 先分别计算 `(SWIR1 + Red)` 与 `(NIR + Blue)`。
5. 对每个像元按 `((SWIR1 + Red) - (NIR + Blue)) / ((SWIR1 + Red) + (NIR + Blue))` 计算 BSI。
6. 对分母接近零的位置按安全除法逻辑处理。
7. 输出为单波段浮点栅格。

## 输出说明

- 计算公式：`((SWIR1 + Red) - (NIR + Blue)) / ((SWIR1 + Red) + (NIR + Blue))`
- 一般可与 NDVI、NDBI 联合判读

## 结果解释

- 较高值通常更容易对应裸地、低覆盖区或部分高反射地表。
- 与建筑和干燥区域可能存在一定混淆，建议结合其他指数共同判断。

## 适用场景

- 裸土提取、低植被覆盖分析和地表背景判读。

## 推荐输出文件名

- `bsi_result.tif`

## 注意事项

- 相关波段号必须落在实际波段范围内
- 结果元数据会返回最小值、最大值以及实际使用的波段号
