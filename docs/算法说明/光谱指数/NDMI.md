# 光谱指数 / NDMI

## 功能说明

根据近红外与短波红外 1 波段计算归一化水分指数 NDMI，常用于植被含水量和地表湿度分析。

## 输入与输出

- 输入：多波段栅格
- 输出：NDMI 栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入多波段影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出 NDMI 结果 |
| 近红外波段 | 整数 | 否 | `4` | 近红外波段序号 |
| 短波红外1波段 | 整数 | 否 | `5` | 短波红外1波段序号 |

## 算法内容

NDMI 通过比较近红外与短波红外 1 的差异来表达植被或地表含水状况。当前实现按经典 NDMI 公式逐像元计算，不引入额外平滑或分级步骤，输出的是单波段浮点指数图。

## 处理流程

1. 打开输入多波段栅格。
2. 检查近红外和短波红外 1 波段号是否有效。
3. 读取近红外与短波红外 1 波段数据。
4. 对每个像元按 `(NIR - SWIR1) / (NIR + SWIR1)` 计算 NDMI。
5. 对分母接近零的位置按安全除法逻辑处理。
6. 将结果写出为单波段浮点栅格。

## 输出说明

- 计算公式：`(NIR - SWIR1) / (NIR + SWIR1)`
- 结果越高通常表示水分越充足

## 结果解释

- 值较高通常表示植被或地表含水状况更明显。
- 值较低或为负时，往往表示更干燥或非湿润目标。

## 适用场景

- 植被含水量、湿度变化和干旱敏感分析。

## 推荐输出文件名

- `ndmi_result.tif`

## 注意事项

- 近红外和短波红外 1 波段号必须落在实际波段范围内
- 结果元数据会返回最小值、最大值以及实际使用的波段号
