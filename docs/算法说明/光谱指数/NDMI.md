# 光谱指数 / NDMI

## 功能说明

根据近红外与短波红外 1 波段计算归一化水分指数 NDMI，常用于植被含水量和地表湿度分析。

## 输入与输出

- 输入：多波段栅格
- 输出：NDMI 栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入多波段影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出 NDMI 结果 |
| 近红外波段 | 整数 | 否 | `4` | 近红外波段序号 |
| 短波红外1波段 | 整数 | 否 | `5` | 短波红外1波段序号 |

## 输出说明

- 计算公式：`(NIR - SWIR1) / (NIR + SWIR1)`
- 结果越高通常表示水分越充足

## 推荐输出文件名

- `ndmi_result.tif`

## 注意事项

- 近红外和短波红外 1 波段号必须落在实际波段范围内
- 结果元数据会返回最小值、最大值以及实际使用的波段号
