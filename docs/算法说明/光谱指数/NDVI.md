# 光谱指数 / NDVI

## 功能说明

根据红光与近红外波段计算 NDVI。

## 输入与输出

- 输入：多波段栅格
- 输出：NDVI 栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入多波段影像。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出 NDVI。 |
| 红光波段 | 整数 | 否 | `3` | 红光波段序号。 |
| 近红外波段 | 整数 | 否 | `4` | 近红外波段序号。 |

## 算法内容

NDVI 是最常用的植被指数之一，用红光与近红外两个波段的差异来衡量植被活跃程度。植被通常会强烈吸收红光、较强反射近红外，因此健康植被区域往往会得到更高的 NDVI 值。

当前实现按经典公式逐像元计算，不引入额外平滑、重缩放或分类步骤。它输出的是一个单波段浮点指数图，用来表达不同位置的相对植被状况，而不是直接输出植被类别。

## 处理流程

1. 打开输入多波段栅格。
2. 检查红光波段和近红外波段编号是否在实际波段范围内。
3. 分别读取红光波段和近红外波段的数据。
4. 对每个像元按 `(NIR - Red) / (NIR + Red)` 计算 NDVI。
5. 在分母接近零或不适合直接相除的位置，按安全除法逻辑处理，避免非法数值直接破坏结果。
6. 将计算出的 NDVI 写出为单波段浮点栅格，并记录结果的最小值、最大值以及实际使用的波段号。

## 输出说明

- 输出结果建议使用 `.tif`。
- 计算公式为 `(NIR - Red) / (NIR + Red)`。
- 理论上 NDVI 常见范围在 `-1` 到 `1`，实际结果会受输入数据值范围影响。

## 结果解释

- 值越接近正高值，通常说明植被覆盖更明显、活跃度更高。
- 接近零的区域，往往表示裸地、建筑或植被较弱区域。
- 明显负值通常对应水体、阴影或某些非植被目标。
- 实际判读时应结合传感器类型、波段定义、预处理状态和场景特征，不应只凭单一阈值机械分割。

## 适用场景

- 植被覆盖快速识别。
- 农情、林地、生长状态等基础遥感分析。
- 作为后续阈值分割、分级制图或时序对比的输入指数。

## 推荐输出文件名

- `ndvi_result.tif`
- `植被指数_ndvi.tif`

## 注意事项

- 红光和近红外波段号都必须落在实际波段范围内。
- 结果元数据返回 NDVI 最小值、最大值和实际使用的波段号。
