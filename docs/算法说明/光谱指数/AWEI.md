# 光谱指数 / AWEI

## 功能说明

根据绿光、近红外、短波红外 1 和短波红外 2 波段计算 AWEI，适合阴影环境下的水体增强。

## 输入与输出
- 输入：多波段栅格
- 输出：AWEI 栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入多波段影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出 AWEI 结果 |
| 绿光波段 | 整数 | 否 | `2` | 绿光波段序号 |
| 近红外波段 | 整数 | 否 | `4` | 近红外波段序号 |
| 短波红外1波段 | 整数 | 否 | `5` | SWIR1 波段序号 |
| 短波红外2波段 | 整数 | 否 | `6` | SWIR2 波段序号 |

## 算法内容

AWEI 用多波段线性组合增强水体响应，尤其适合在阴影、暗背景等复杂条件下做水体辅助识别。当前实现按固定公式逐像元计算，输出单波段浮点结果图。它不是归一化指数，因此值域不受 `-1` 到 `1` 限制。

## 处理流程

1. 打开输入多波段栅格。
2. 检查绿光、近红外、短波红外 1 和短波红外 2 波段号是否有效。
3. 读取四个参与计算的波段。
4. 对每个像元按 `4 * (Green - SWIR1) - (0.25 * NIR + 2.75 * SWIR2)` 计算 AWEI。
5. 将结果直接写出为单波段浮点栅格。

## 输出说明

- 计算公式：`4 * (Green - SWIR1) - (0.25 * NIR + 2.75 * SWIR2)`
- 输出元数据会返回指数最小值、最大值

## 结果解释

- 较高值通常更容易对应水体。
- 由于不是归一化指数，具体阈值要结合数据类型、场景和样例试算确定。

## 适用场景

- 阴影环境下的水体增强、水体阈值试验和辅助判读。

## 推荐输出文件名
- `awei_result.tif`

## 注意事项

- AWEI 的值域不一定落在 `-1` 到 `1`，判读时更适合结合阈值或对比样例
