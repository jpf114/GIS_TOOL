# 光谱指数 / OSAVI

## 功能说明

根据红光与近红外波段计算优化土壤调节植被指数 OSAVI，适合减少土壤背景对植被指数的影响。

## 输入与输出

- 输入：多波段栅格
- 输出：OSAVI 栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入多波段影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出 OSAVI 结果 |
| 红光波段 | 整数 | 否 | `3` | 红光波段序号 |
| 近红外波段 | 整数 | 否 | `4` | 近红外波段序号 |

## 算法内容

OSAVI 可以看作对 SAVI 的固定参数优化版本，用更简化的方式抑制土壤背景影响。当前实现按固定 OSAVI 公式逐像元计算，输出单波段浮点指数图。

## 处理流程

1. 打开输入多波段栅格。
2. 检查红光和近红外波段号是否有效。
3. 读取红光与近红外波段数据。
4. 对每个像元按 `1.16 * (NIR - Red) / (NIR + Red + 0.16)` 计算 OSAVI。
5. 对分母接近零的位置按安全除法逻辑处理。
6. 输出为单波段浮点栅格。

## 输出说明

- 计算公式：`1.16 * (NIR - Red) / (NIR + Red + 0.16)`
- 相比 SAVI 固定采用常见的优化参数

## 结果解释

- 较高值通常表示植被响应更强。
- 与 SAVI 相比，它不需要额外调 `L` 参数，更适合统一口径快速处理。

## 适用场景

- 希望在减弱土壤背景影响的同时保持参数简单的植被分析任务。

## 推荐输出文件名

- `osavi_result.tif`

## 注意事项

- 红光和近红外波段号必须落在实际波段范围内
- 结果元数据会返回最小值、最大值以及实际使用的波段号
