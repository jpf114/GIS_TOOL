# 光谱指数 / EVI

## 功能说明

根据蓝光、红光和近红外波段计算 EVI，适合在高植被覆盖场景下减弱土壤与大气背景影响。

## 输入与输出
- 输入：多波段栅格
- 输出：EVI 栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入多波段影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出 EVI 结果 |
| 蓝光波段 | 整数 | 否 | `1` | 蓝光波段序号 |
| 红光波段 | 整数 | 否 | `3` | 红光波段序号 |
| 近红外波段 | 整数 | 否 | `4` | 近红外波段序号 |
| G 参数 | 浮点数 | 否 | `2.5` | EVI 增益系数 |
| C1 参数 | 浮点数 | 否 | `6.0` | 红光校正系数 |
| C2 参数 | 浮点数 | 否 | `7.5` | 蓝光校正系数 |
| L 参数 | 浮点数 | 否 | `1.0` | 背景校正项 |

## 算法内容

EVI 在植被指数计算中同时考虑增益、大气修正和背景修正，常用于高植被覆盖场景下比 NDVI 更稳定地表达植被活性。当前实现按给定参数逐像元计算，输出单波段浮点指数图。

## 处理流程

1. 打开输入多波段栅格。
2. 检查蓝光、红光和近红外波段号是否有效。
3. 读取蓝光、红光和近红外波段数据。
4. 根据 `G`、`C1`、`C2`、`L` 组装 EVI 分母与分子。
5. 对每个像元按 `G * (NIR - Red) / (NIR + C1 * Red - C2 * Blue + L)` 计算 EVI。
6. 对分母接近零的位置按安全除法逻辑处理。
7. 输出为单波段浮点栅格。

## 输出说明

- 计算公式：`G * (NIR - Red) / (NIR + C1 * Red - C2 * Blue + L)`
- 输出元数据会返回指数最小值、最大值

## 结果解释

- 较高值通常表示植被更旺盛。
- 与 NDVI 相比，EVI 在高覆盖植被和复杂背景下往往更稳定，但对蓝光质量和参数设置更敏感。

## 适用场景

- 高植被覆盖区域分析、农林监测和需要增强背景修正的场景。

## 推荐输出文件名
- `evi_result.tif`

## 注意事项

- 蓝光、红光、近红外波段号都必须落在实际波段范围内
- 如果输入影像波段排列不同，需要先确认传感器的真实波段顺序
