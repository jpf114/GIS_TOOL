# 光谱指数 / GNDVI

## 功能说明

根据绿光与近红外波段计算 GNDVI，适合描述植被叶绿素活性与氮素变化。

## 输入与输出
- 输入：多波段栅格
- 输出：GNDVI 栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入多波段影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出 GNDVI 结果 |
| 绿光波段 | 整数 | 否 | `2` | 绿光波段序号 |
| 近红外波段 | 整数 | 否 | `4` | 近红外波段序号 |

## 算法内容

GNDVI 用绿光替代 NDVI 中的红光，常用于表达植被叶绿素活性和生长状态差异。当前实现按标准公式逐像元计算，输出单波段浮点指数图。

## 处理流程

1. 打开输入栅格并检查绿光、近红外波段号。
2. 读取绿光和近红外波段数据。
3. 对每个像元按 `(NIR - Green) / (NIR + Green)` 计算 GNDVI。
4. 对分母接近零的位置按安全除法逻辑处理。
5. 输出为单波段浮点栅格。

## 输出说明

- 计算公式：`(NIR - Green) / (NIR + Green)`
- 输出元数据会返回指数最小值、最大值

## 结果解释

- 值越高通常表示植被活性越强。
- 相比 NDVI，它对叶绿素变化更敏感，但仍需结合具体传感器和场景判读。

## 适用场景

- 精细植被长势分析、农情监测和叶绿素变化观察。

## 推荐输出文件名
- `gndvi_result.tif`

## 注意事项

- 如果绿光波段选择错误，结果会明显偏离植被实际分布
