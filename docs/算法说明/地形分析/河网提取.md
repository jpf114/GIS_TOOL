# 地形分析 / 河网提取

## 功能说明

基于汇流累积量阈值提取河网栅格，输出河网掩膜结果。

## 输入与输出

- 输入：DEM 栅格
- 输出：河网提取结果栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | DEM 输入 |
| 输出文件 | 文件路径 | 是 | 无 | 输出河网栅格 |
| 波段号 | 整数 | 否 | `1` | DEM 波段 |
| z_factor | 浮点数 | 否 | `1.0` | 高程缩放系数 |
| accum_threshold | 浮点数 | 否 | `10.0` | 汇流累积阈值，大于等于该值的位置视为河网 |

## 算法内容

河网提取当前采用“先汇流累积，再阈值分割”的方式完成。也就是说，算法不会直接从 DEM 几何形态中拟合河道，而是先根据 D8 水流模型得到每个像元的上游汇集量，再把达到阈值的位置标记为河网。

## 处理流程

1. 打开输入 DEM 并读取指定波段。
2. 校验 `accum_threshold` 必须大于 0。
3. 先复用汇流累积流程，生成当前 DEM 的汇流累积结果。
4. 读取汇流累积栅格。
5. 将累积值大于等于 `accum_threshold` 的像元标记为河网。
6. 生成二值结果栅格：河网位置写为 1，非河网位置写为 0。
7. 若存在 NoData，则恢复这些位置为 NoData。
8. 输出河网掩膜栅格。

## 注意事项

- 当前实现会先计算汇流累积，再按阈值转成二值河网结果。
- `accum_threshold` 必须大于 `0`。
- 阈值越大，提取出的河网通常越稀疏。

## 输出说明

- 输出结果为单波段二值栅格。
- 河网像元值通常为 `1`，非河网像元通常为 `0`。

## 结果解释

- 阈值较小，会得到更密集的河网结果。
- 阈值较大，则更偏向保留主沟道和主要汇流通道。

## 适用场景

- DEM 河网快速提取、水文骨架生成和流域分析辅助。

## 推荐输出文件名

- `terrain_stream_extract_result.tif`
