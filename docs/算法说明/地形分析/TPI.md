# 地形分析 / TPI

## 功能说明

按 `3x3` 邻域计算地形位置指数，用于描述像元相对周边地形的高低关系。

## 输入与输出

- 输入：DEM 栅格
- 输出：TPI 结果栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | DEM 输入 |
| 输出文件 | 文件路径 | 是 | 无 | 输出 TPI 栅格 |
| 波段号 | 整数 | 否 | `1` | DEM 波段 |
| z_factor | 浮点数 | 否 | `1.0` | 高程缩放系数 |

## 算法内容

TPI 用于描述某个像元相对于周边邻域是偏高还是偏低。当前实现使用固定 `3x3` 邻域，先求周边 8 邻元的平均高程，再用中心像元减去该平均值，得到局部地形位置指数。

## 处理流程

1. 打开输入 DEM 并读取指定波段。
2. 按 `z_factor` 对高程做可选缩放。
3. 以固定 `3x3` 邻域计算局部均值。
4. 从邻域总和中扣除中心像元，得到周边 8 个邻元的平均高程。
5. 用中心像元减去周边平均高程，得到 TPI。
6. 若存在 NoData，则把这些位置恢复为原 NoData 值。
7. 输出为单波段浮点栅格。

## 注意事项

- 当前实现使用固定 `3x3` 邻域。
- 输出值大于 `0` 通常表示相对周边偏高，小于 `0` 表示相对周边偏低。

## 处理流程说明补充

- 当前实现窗口大小固定为 `3x3`，不支持在参数中修改邻域范围。

## 输出说明

- 输出结果为栅格文件。
- 建议优先使用 `.tif` 保存，便于后续在 CLI、GUI 或 GIS 软件中继续处理。

## 结果解释

- 值大于 0，通常表示该像元相对周边地形偏高，如脊部、凸起。
- 值小于 0，通常表示该像元相对周边偏低，如谷地、凹地。

## 适用场景

- 局部地形位置分析、脊谷识别和地貌辅助分类。

## 推荐输出文件名

- `terrain_tpi_result.tif`
