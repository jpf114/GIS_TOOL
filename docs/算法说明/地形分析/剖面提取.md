# 地形分析 / 剖面提取

## 功能说明

沿给定折线路径采样 DEM 高程，并导出剖面 CSV。

## 输入与输出

- 输入：DEM 栅格
- 输出：剖面采样结果，建议 `.csv`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | DEM 输入 |
| 输出文件 | 文件路径 | 是 | 无 | 输出剖面 CSV |
| 波段号 | 整数 | 否 | `1` | DEM 波段 |
| profile_path | 字符串 | 是 | 无 | 剖面路径，格式为 `x1,y1;x2,y2;...` |

## 算法内容

剖面提取用于沿给定折线路径从 DEM 中连续采样高程，生成可直接用于绘制纵剖面曲线的 CSV 结果。当前实现不是按像素路径输入，而是按地图坐标路径输入，再借助栅格地理变换把路径点转换到像素位置进行采样。

## 处理流程

1. 打开输入 DEM 并检查指定波段。
2. 读取栅格地理变换，并计算其逆变换。
3. 解析 `profile_path`，将输入字符串拆成折线顶点列表。
4. 根据栅格分辨率确定沿路径的采样步长。
5. 对折线的每一段逐步插值，生成沿线采样点。
6. 对每个采样点，将地图坐标转换为像素行列号。
7. 从 DEM 指定波段中读取该位置高程值。
8. 按 `distance,x,y,elevation` 的固定列格式写入 CSV。
9. 在整条路径采样完成后输出结果文件。

## 输出说明

输出 CSV 表头固定为：

- `distance`
- `x`
- `y`
- `elevation`

## 结果解释

- `distance` 表示沿折线路径累计的距离。
- `x`、`y` 表示采样点的地图坐标。
- `elevation` 表示该采样点从 DEM 中读取到的高程值。

## 适用场景

- 地形剖面绘制、道路或管线沿线高程分析。
- 坡面变化和地形起伏的定量检查。

## 注意事项

- `profile_path` 至少需要两个点。
- 输出文件应使用 `.csv`。
- 路径坐标使用栅格空间参考下的地图坐标，而不是像素坐标。

## 推荐输出文件名

- `terrain_profile_result.csv`
