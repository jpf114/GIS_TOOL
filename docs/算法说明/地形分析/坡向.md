# 地形分析 / 坡向

## 功能说明

根据 DEM 计算坡向栅格。

## 输入与输出

- 输入：DEM 栅格
- 输出：坡向结果栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | DEM 输入 |
| 输出文件 | 文件路径 | 是 | 无 | 输出坡向栅格 |
| 波段号 | 整数 | 否 | `1` | DEM 波段 |
| z_factor | 浮点数 | 否 | `1.0` | 高程缩放系数 |

## 推荐输出文件名

- `terrain_aspect_result.tif`
