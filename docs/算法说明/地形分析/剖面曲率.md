# 地形分析 / 剖面曲率

## 功能说明

按 `3x3` 邻域估算沿坡向的剖面曲率，用于描述坡面顺坡方向的凹凸变化。

## 输入与输出

- 输入：DEM 栅格
- 输出：剖面曲率结果栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | DEM 输入 |
| 输出文件 | 文件路径 | 是 | 无 | 输出剖面曲率栅格 |
| 波段号 | 整数 | 否 | `1` | DEM 波段 |
| z_factor | 浮点数 | 否 | `1.0` | 高程缩放系数 |

## 注意事项

- 当前实现基于局部一阶、二阶导数近似，适合轻量分析与快速预览
- 地形较平坦时结果会自动趋近于 `0`

## 推荐输出文件名

- `terrain_profile_curvature_result.tif`
