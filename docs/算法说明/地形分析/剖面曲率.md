# 地形分析 / 剖面曲率

## 功能说明

按 `3x3` 邻域估算沿坡向的剖面曲率，用于描述坡面顺坡方向的凹凸变化。

## 输入与输出

- 输入：DEM 栅格
- 输出：剖面曲率结果栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | DEM 输入 |
| 输出文件 | 文件路径 | 是 | 无 | 输出剖面曲率栅格 |
| 波段号 | 整数 | 否 | `1` | DEM 波段 |
| z_factor | 浮点数 | 否 | `1.0` | 高程缩放系数 |

## 算法内容

剖面曲率用于描述沿坡向的凹凸变化趋势。当前实现先在固定 `3x3` 邻域上估计一阶和二阶导数，再按剖面曲率公式逐像元计算结果。

## 处理流程

1. 打开输入 DEM 并读取指定波段。
2. 按 `z_factor` 对高程进行缩放。
3. 通过局部导数近似计算一阶导数和二阶导数。
4. 对每个像元计算坡度平方项。
5. 若坡度过小，则该像元结果直接写为 0。
6. 否则按剖面曲率公式计算沿坡向的弯曲变化。
7. 若存在 NoData，则恢复这些位置的 NoData 值。
8. 输出为单波段浮点栅格。

## 注意事项

- 当前实现基于局部一阶、二阶导数近似，适合轻量分析与快速预览
- 地形较平坦时结果会自动趋近于 `0`

## 输出说明

- 输出结果为单波段浮点栅格。
- 建议优先使用 `.tif` 保存，便于后续在 CLI、GUI 或 GIS 软件中继续处理。

## 结果解释

- 可用于辅助判断坡面在顺坡方向上的凹形或凸形趋势。
- 平坦区域或坡度极小区域结果会自动压到 0，避免不稳定值。

## 适用场景

- 坡面形态分析、侵蚀趋势辅助判断和与平面曲率联合解读。

## 推荐输出文件名

- `terrain_profile_curvature_result.tif`
