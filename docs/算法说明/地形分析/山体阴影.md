# 地形分析 / 山体阴影

## 功能说明

根据 DEM、太阳方位角和高度角生成山体阴影效果图。

## 输入与输出

- 输入：DEM 栅格
- 输出：山体阴影栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | DEM 输入 |
| 输出文件 | 文件路径 | 是 | 无 | 输出阴影结果 |
| 波段号 | 整数 | 否 | `1` | DEM 波段 |
| z_factor | 浮点数 | 否 | `1.0` | 高程缩放系数 |
| azimuth | 浮点数 | 否 | `315` | 太阳方位角，范围 `[0, 360]` |
| altitude | 浮点数 | 否 | `45` | 太阳高度角，范围 `[0, 90]` |

## 算法内容

山体阴影用于模拟在给定太阳方位角和高度角条件下，地形表面的受光明暗效果。当前实现直接调用 GDAL 的 hillshade 流程，根据 DEM 生成阴影栅格，适合地形可视化和地貌辅助判读。

## 处理流程

1. 打开输入 DEM 并检查指定波段。
2. 将波段号、`z_factor`、太阳方位角 `azimuth` 和太阳高度角 `altitude` 组装为处理参数。
3. 调用 GDAL 的山体阴影生成流程。
4. 由 GDAL 根据 DEM 邻域坡度和光照方向生成阴影结果。
5. 将结果输出为栅格文件。

## 输出说明

- 输出结果为栅格文件。
- 建议优先使用 `.tif` 保存，便于后续在 CLI、GUI 或 GIS 软件中继续处理。

## 结果解释

- 亮部通常表示更直接受光的坡面，暗部表示背光或遮蔽更明显的区域。
- 输出主要用于显示和辅助判读，不直接表示真实反射率或物理量。

## 适用场景

- DEM 可视化、地貌展示、制图底图增强和坡面朝向辅助分析。

## 推荐输出文件名

- `terrain_hillshade_result.tif`

## 注意事项

- 建议输入具有明确高程含义的 DEM 数据。
- 涉及距离、面积或坡度解释时，应确认坐标系与高程单位一致。
