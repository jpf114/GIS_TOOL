# 地形分析 / 平面曲率

## 功能说明

按 `3x3` 邻域估算垂直坡向的平面曲率，用于描述等高线方向的收敛与发散趋势。

## 输入与输出

- 输入：DEM 栅格
- 输出：平面曲率结果栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | DEM 输入 |
| 输出文件 | 文件路径 | 是 | 无 | 输出平面曲率栅格 |
| 波段号 | 整数 | 否 | `1` | DEM 波段 |
| z_factor | 浮点数 | 否 | `1.0` | 高程缩放系数 |

## 算法内容

平面曲率用于描述等高线方向上的收敛与发散趋势。当前实现先在固定 `3x3` 邻域上估计一阶、二阶地形导数，再按平面曲率公式逐像元计算结果。

## 处理流程

1. 打开输入 DEM 并读取指定波段。
2. 按 `z_factor` 对高程进行缩放。
3. 通过 Sobel 等局部导数近似计算一阶导数和二阶导数。
4. 对每个像元计算坡度平方项。
5. 若局部坡度过小，则该像元结果直接写为 0。
6. 否则按平面曲率公式计算结果。
7. 若存在 NoData，则恢复这些位置的 NoData 值。
8. 输出为单波段浮点栅格。

## 注意事项

- 当前实现基于局部一阶、二阶导数近似，适合轻量分析与快速预览
- 可结合剖面曲率一起判读坡面形态

## 输出说明

- 输出结果为单波段浮点栅格。
- 建议优先使用 `.tif` 保存，便于后续在 CLI、GUI 或 GIS 软件中继续处理。

## 结果解释

- 不同符号和数值大小可用于辅助判断地形收敛、发散和侧向形态变化。
- 平坦区域会因为坡度过小而自动趋近于 0。

## 适用场景

- 坡面形态分析、汇流趋势辅助判断和与剖面曲率联合判读。

## 推荐输出文件名

- `terrain_plan_curvature_result.tif`
