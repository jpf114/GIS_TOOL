# 地形分析 / 填洼

## 功能说明

按局部邻域填平 DEM 中的洼地，为后续水文分析做预处理。

## 输入与输出

- 输入：DEM 栅格
- 输出：填洼后的 DEM 栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | DEM 输入 |
| 输出文件 | 文件路径 | 是 | 无 | 输出填洼结果 |
| 波段号 | 整数 | 否 | `1` | DEM 波段 |
| z_factor | 浮点数 | 否 | `1.0` | 高程缩放系数 |

## 算法内容

填洼用于消除 DEM 中阻断水流分析的局部洼地，是很多水文分析前的预处理步骤。当前实现采用局部迭代方式，在 `3x3` 邻域上逐步抬升低洼像元，使其不再低于周边最小邻值。

## 处理流程

1. 打开输入 DEM 并读取指定波段。
2. 按 `z_factor` 对高程进行缩放。
3. 构建 NoData 掩膜，避免无效像元参与迭代。
4. 以当前高程为初始结果，在 `3x3` 邻域上反复求邻域最小值。
5. 将每轮结果更新为“原值与邻域最小值中的较大者”，逐步填平局部洼地。
6. 如果某一轮不再发生变化，则提前结束迭代。
7. 恢复 NoData 位置并输出结果 DEM。

## 输出说明

- 输出结果为栅格文件。
- 建议优先使用 `.tif` 保存，便于后续在 CLI、GUI 或 GIS 软件中继续处理。

## 结果解释

- 输出是一个经过填洼处理的 DEM，而不是水流结果本身。
- 原本封闭或阻断流动的小型洼地会被抬平，更适合后续流向、汇流和流域分析。

## 适用场景

- 流向、汇流累积、河网提取、流域划分前的预处理。

## 推荐输出文件名

- `terrain_fill_sinks_result.tif`

## 注意事项

- 建议输入具有明确高程含义的 DEM 数据。
- 涉及距离、面积或坡度解释时，应确认坐标系与高程单位一致。
