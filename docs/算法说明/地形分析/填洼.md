# 地形分析 / 填洼

## 功能说明

按局部邻域填平 DEM 中的洼地，为后续水文分析做预处理。

## 输入与输出

- 输入：DEM 栅格
- 输出：填洼后的 DEM 栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | DEM 输入 |
| 输出文件 | 文件路径 | 是 | 无 | 输出填洼结果 |
| 波段号 | 整数 | 否 | `1` | DEM 波段 |
| z_factor | 浮点数 | 否 | `1.0` | 高程缩放系数 |

## 输出说明

- 输出结果为栅格文件。
- 建议优先使用 `.tif` 保存，便于后续在 CLI、GUI 或 GIS 软件中继续处理。

## 推荐输出文件名

- `terrain_fill_sinks_result.tif`

## 注意事项

- 建议输入具有明确高程含义的 DEM 数据。
- 涉及距离、面积或坡度解释时，应确认坐标系与高程单位一致。
