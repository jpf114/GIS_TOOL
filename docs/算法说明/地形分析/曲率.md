# 地形分析 / 曲率

## 功能说明

按 `3x3` 邻域计算 DEM 表面曲率近似值，用于快速描述地表弯曲变化。

## 输入与输出

- 输入：DEM 栅格
- 输出：曲率结果栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | DEM 输入 |
| 输出文件 | 文件路径 | 是 | 无 | 输出曲率栅格 |
| 波段号 | 整数 | 否 | `1` | DEM 波段 |
| z_factor | 浮点数 | 否 | `1.0` | 高程缩放系数 |

## 算法内容

曲率用于描述地表弯曲变化趋势。当前实现采用局部拉普拉斯近似，在固定 `3x3` 邻域上估计地表整体弯曲程度，适合快速地形形态预览和轻量分析。

## 处理流程

1. 打开输入 DEM 并读取指定波段。
2. 按 `z_factor` 对高程进行缩放。
3. 在固定 `3x3` 邻域上对高程执行二阶导数近似。
4. 使用局部拉普拉斯结果作为曲率近似值。
5. 若存在 NoData，则恢复这些位置的 NoData 值。
6. 输出为单波段浮点栅格。

## 注意事项

- 当前实现基于局部拉普拉斯近似，适合轻量分析和快速预览。
- 若 DEM 噪声较大，结果会更敏感，必要时可先做平滑处理。

## 输出说明

- 输出结果为单波段浮点栅格。
- 建议优先使用 `.tif` 保存，便于后续在 CLI、GUI 或 GIS 软件中继续处理。

## 结果解释

- 正负值可用于区分不同弯曲趋势，但具体解释应结合项目使用口径。
- 值幅度越大，通常表示局部弯曲变化越明显。

## 适用场景

- 地表弯曲特征快速分析、地貌辅助识别和后续曲率分解前的预览。

## 推荐输出文件名

- `terrain_curvature_result.tif`
