# 地形分析 / 库容计算

## 功能说明

按指定水位计算 DEM 蓄水深度，并统计淹没面积与库容。

## 输入与输出

- 输入：DEM 栅格
- 输出：蓄水深度结果栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | DEM 输入 |
| 输出文件 | 文件路径 | 是 | 无 | 输出蓄水深度栅格 |
| 波段号 | 整数 | 否 | `1` | DEM 波段 |
| water_level | 浮点数 | 是 | 无 | 目标水位 |

## 输出说明

结果元数据中会附带：

- `reservoir_area`
- `reservoir_volume`
- `water_level`

## 注意事项

- 当前输出栅格表示 `max(water_level - elevation, 0)`。
- `water_level` 应与 DEM 高程单位保持一致。
- 淹没面积与库容基于像元面积直接累计。

## 推荐输出文件名

- `terrain_reservoir_volume_result.tif`
