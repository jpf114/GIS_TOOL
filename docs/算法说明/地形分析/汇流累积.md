# 地形分析 / 汇流累积

## 功能说明

沿 D8 主流向累计每个像元的上游像元数量。

## 输入与输出

- 输入：DEM 栅格
- 输出：汇流累积栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | DEM 输入 |
| 输出文件 | 文件路径 | 是 | 无 | 输出汇流结果 |
| 波段号 | 整数 | 否 | `1` | DEM 波段 |
| z_factor | 浮点数 | 否 | `1.0` | 高程缩放系数 |

## 算法内容

汇流累积用于统计每个像元能够接收多少上游贡献，是河网提取和流域分析的重要基础。当前实现先按 D8 规则建立流向矩阵，再根据上下游依赖关系，把上游像元数量逐步累加到下游。

## 处理流程

1. 打开输入 DEM 并读取指定波段。
2. 按 `z_factor` 对高程进行缩放。
3. 先按 D8 规则生成流向矩阵。
4. 统计每个像元被多少上游像元指向，建立入度关系。
5. 将每个有效像元初始化为 1，表示至少包含自身贡献。
6. 从没有上游来水的像元开始，按拓扑顺序向下游累加贡献值。
7. 累加结束后得到每个像元的汇流累积量。
8. 恢复 NoData 位置并输出结果栅格。

## 输出说明

- 输出结果为栅格文件。
- 建议优先使用 `.tif` 保存，便于后续在 CLI、GUI 或 GIS 软件中继续处理。

## 结果解释

- 值越大，表示汇入该像元的上游范围越大。
- 高值区域通常更接近主沟谷或潜在河道。

## 适用场景

- 河网提取、流域分析和汇流路径识别。

## 推荐输出文件名

- `terrain_flow_accumulation_result.tif`

## 注意事项

- 建议输入具有明确高程含义的 DEM 数据。
- 涉及距离、面积或坡度解释时，应确认坐标系与高程单位一致。
