# 地形分析 / 填挖方

## 功能说明

比较当前 DEM 与参考 DEM，输出高差栅格，并统计填方、挖方和净体积。

## 输入与输出

- 输入：
  - 输入文件：当前 DEM
  - 参考文件：参考 DEM
- 输出：高差结果栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 参考文件 | 文件路径 | 是 | 无 | 参考 DEM |
| 输入文件 | 文件路径 | 是 | 无 | 当前 DEM |
| 输出文件 | 文件路径 | 是 | 无 | 输出高差栅格 |
| 波段号 | 整数 | 否 | `1` | 使用的 DEM 波段 |

## 算法内容

填挖方用于比较两个 DEM 之间的高程差异，并进一步估算填方量、挖方量和净体积。当前实现把“输入 DEM - 参考 DEM”作为高差结果，并按像元面积对正负高差分别积分。

## 处理流程

1. 打开输入 DEM 和参考 DEM，并检查指定波段。
2. 检查两者栅格尺寸是否一致。
3. 如果两者都具有地理变换信息，则进一步检查地理变换是否一致。
4. 读取两个 DEM 的对应波段。
5. 逐像元计算高差：`输入 DEM - 参考 DEM`。
6. 根据栅格地理变换计算单个像元面积。
7. 对高差大于 0 的部分累计填方体积，对高差小于 0 的部分累计挖方体积。
8. 输出高差栅格，并在元数据中记录填方、挖方和净体积。

## 输出说明

结果元数据中会附带：

- `fill_volume`
- `cut_volume`
- `net_volume`

## 结果解释

- 输出栅格中的正值表示输入 DEM 高于参考 DEM，负值表示输入 DEM 低于参考 DEM。
- `fill_volume` 表示正高差部分累计得到的填方体积。
- `cut_volume` 表示负高差部分累计得到的挖方体积。
- `net_volume` 为填方减去挖方后的净结果。

## 适用场景

- 土方量分析、施工前后地形对比和场地整形评估。

## 注意事项

- 输入 DEM 与参考 DEM 的尺寸必须一致。
- 若两者具有地理变换信息，当前实现要求地理变换一致。
- 输出值为 `输入 DEM - 参考 DEM`。

## 推荐输出文件名

- `terrain_cut_fill_result.tif`
