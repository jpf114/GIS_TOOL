# 地形分析 / 填挖方

## 功能说明

比较当前 DEM 与参考 DEM，输出高差栅格，并统计填方、挖方和净体积。

## 输入与输出

- 输入：
  - 输入文件：当前 DEM
  - 参考文件：参考 DEM
- 输出：高差结果栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 参考文件 | 文件路径 | 是 | 无 | 参考 DEM |
| 输入文件 | 文件路径 | 是 | 无 | 当前 DEM |
| 输出文件 | 文件路径 | 是 | 无 | 输出高差栅格 |
| 波段号 | 整数 | 否 | `1` | 使用的 DEM 波段 |

## 输出说明

结果元数据中会附带：

- `fill_volume`
- `cut_volume`
- `net_volume`

## 注意事项

- 输入 DEM 与参考 DEM 的尺寸必须一致。
- 若两者具有地理变换信息，当前实现要求地理变换一致。
- 输出值为 `输入 DEM - 参考 DEM`。

## 推荐输出文件名

- `terrain_cut_fill_result.tif`
