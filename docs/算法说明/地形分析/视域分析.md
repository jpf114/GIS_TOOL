# 地形分析 / 视域分析

## 功能说明

以单个观察点为中心，计算 DEM 上的可视域范围。

## 输入与输出

- 输入：DEM 栅格
- 输出：视域结果栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | DEM 输入 |
| 输出文件 | 文件路径 | 是 | 无 | 输出视域结果 |
| 波段号 | 整数 | 否 | `1` | DEM 波段 |
| observer_x | 浮点数 | 是 | 无 | 观察点 X 坐标 |
| observer_y | 浮点数 | 是 | 无 | 观察点 Y 坐标 |
| observer_height | 浮点数 | 否 | `2.0` | 观察点高度，必须大于等于 0 |
| target_height | 浮点数 | 否 | `0.0` | 目标高度，必须大于等于 0 |
| max_distance | 浮点数 | 否 | `0.0` | 最大分析距离，`0` 表示不限 |

## 输出说明

- 输出结果为栅格文件。
- 建议优先使用 `.tif` 保存，便于后续在 CLI、GUI 或 GIS 软件中继续处理。

## 推荐输出文件名

- `terrain_viewshed_result.tif`

## 注意事项

- 建议输入具有明确高程含义的 DEM 数据。
- 涉及距离、面积或坡度解释时，应确认坐标系与高程单位一致。
