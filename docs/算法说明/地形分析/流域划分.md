# 地形分析 / 流域划分

## 功能说明

按 D8 主流向自动生成流域编号栅格。

## 输入与输出

- 输入：DEM 栅格
- 输出：流域编号栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | DEM 输入 |
| 输出文件 | 文件路径 | 是 | 无 | 输出流域结果 |
| 波段号 | 整数 | 否 | `1` | DEM 波段 |
| z_factor | 浮点数 | 否 | `1.0` | 高程缩放系数 |

## 算法内容

流域划分用于根据水流去向，把 DEM 中所有像元归属到不同下游出口。当前实现采用 D8 流向模型，先确定主流向，再从出口位置反向回填上游归属，最终生成流域编号栅格。

## 处理流程

1. 打开输入 DEM 并读取指定波段。
2. 按 `z_factor` 对高程进行缩放。
3. 先按 D8 规则生成流向矩阵。
4. 统计每个像元的入度关系。
5. 识别没有有效下游、或流向指向边界外的像元，将其作为出口点并赋予新的流域编号。
6. 建立从源头到下游的拓扑顺序。
7. 按拓扑逆序把下游出口编号回填到所有上游像元。
8. 将流域编号结果输出为单波段栅格，并恢复 NoData 位置。

## 输出说明

- 输出结果为栅格文件。
- 建议优先使用 `.tif` 保存，便于后续在 CLI、GUI 或 GIS 软件中继续处理。

## 结果解释

- 输出像元值不是高程，而是流域编号。
- 编号相同的像元表示它们最终汇入同一出口区域。

## 适用场景

- 集水区分析、河流分区和水文单元划分。

## 推荐输出文件名

- `terrain_watershed_result.tif`

## 注意事项

- 建议输入具有明确高程含义的 DEM 数据。
- 涉及距离、面积或坡度解释时，应确认坐标系与高程单位一致。
