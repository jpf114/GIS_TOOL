# 地形分析 / 粗糙度

## 功能说明

按 `3x3` 邻域计算地表粗糙度，输出邻域内最大值与最小值之差。

## 输入与输出

- 输入：DEM 栅格
- 输出：粗糙度结果栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | DEM 输入 |
| 输出文件 | 文件路径 | 是 | 无 | 输出粗糙度栅格 |
| 波段号 | 整数 | 否 | `1` | DEM 波段 |
| z_factor | 浮点数 | 否 | `1.0` | 高程缩放系数 |

## 算法内容

粗糙度用于描述局部区域内最大起伏差异。当前实现使用固定 `3x3` 邻域，通过计算邻域最大值与最小值之差来表达局部地表起伏强弱。

## 处理流程

1. 打开输入 DEM 并读取指定波段。
2. 按 `z_factor` 对高程进行缩放。
3. 在 `3x3` 邻域内分别计算局部最大值和最小值。
4. 用局部最大值减去局部最小值，得到粗糙度结果。
5. 若存在 NoData，则恢复这些位置的 NoData 值。
6. 输出为单波段浮点栅格。

## 注意事项

- 当前实现使用 `3x3` 邻域最大最小高差。
- 输出值越大，通常表示局部地形起伏越强。

## 输出说明

- 输出结果为单波段浮点栅格。
- 建议优先使用 `.tif` 保存，便于后续在 CLI、GUI 或 GIS 软件中继续处理。

## 结果解释

- 值越大，表示该邻域内高差范围越大，地表越不平整。
- 值越小，表示局部地形更平滑。

## 适用场景

- 局部地形起伏分析、粗糙度分级和地形复杂度辅助评价。

## 推荐输出文件名

- `terrain_roughness_result.tif`
