# 地形分析 / 粗糙度

## 功能说明

按 `3x3` 邻域计算地表粗糙度，输出邻域内最大值与最小值之差。

## 输入与输出

- 输入：DEM 栅格
- 输出：粗糙度结果栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | DEM 输入 |
| 输出文件 | 文件路径 | 是 | 无 | 输出粗糙度栅格 |
| 波段号 | 整数 | 否 | `1` | DEM 波段 |
| z_factor | 浮点数 | 否 | `1.0` | 高程缩放系数 |

## 注意事项

- 当前实现使用 `3x3` 邻域最大最小高差。
- 输出值越大，通常表示局部地形起伏越强。

## 推荐输出文件名

- `terrain_roughness_result.tif`
