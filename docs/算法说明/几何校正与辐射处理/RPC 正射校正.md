# 几何校正与辐射处理 / RPC 正射校正

## 功能说明

基于影像自带 RPC 元数据执行轻量正射校正，可选 DEM 参与。

## 输入与输出

- 输入：
  - 带 RPC 元数据的栅格影像
  - 可选 DEM
- 输出：
  - 正射校正结果栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出结果 |
| dst_srs | 字符串 | 是 | 无 | 目标坐标系，如 `EPSG:4326` |
| dem_file | 文件路径 | 否 | 无 | 可选 DEM 文件 |
| rpc_height | 浮点数 | 否 | `0.0` | 不使用 DEM 时的固定高程 |
| resample | 枚举 | 否 | `nearest` | 重采样方式 |

## 输出说明

- 输出结果带目标投影
- 不提供 DEM 时，使用固定高程参与 RPC 解算

## 推荐输出文件名

- `rpc_orthorectify_result.tif`

## 注意事项

- 输入影像必须包含 `RPC` 元数据
- 当前实现基于 GDAL `-rpc` 流程，适合轻量正射预处理
