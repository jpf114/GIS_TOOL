# 几何校正与辐射处理 / DOS 大气校正

## 功能说明

对单波段栅格执行简化暗像元大气校正，并将结果归一化。

## 输入与输出

- 输入：单波段或多波段栅格
- 输出：单波段浮点栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出校正结果 |
| 波段号 | 整数 | 否 | `1` | 参与处理的波段 |
| dark_object_value | 浮点数 | 否 | 自动估计 | 手工指定暗像元值 |

## 输出说明

- 输出结果为非负浮点栅格
- 当亮暗差值可用时，结果会归一化到近似 `0` 到 `1`

## 推荐输出文件名

- `dos_correction_result.tif`

## 注意事项

- 当前实现为简化 DOS，适合基础流程验证与轻量处理
