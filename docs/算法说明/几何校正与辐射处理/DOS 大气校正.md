# 几何校正与辐射处理 / DOS 大气校正

## 功能说明

对单波段栅格执行简化暗像元大气校正，并将结果归一化。

## 输入与输出

- 输入：单波段或多波段栅格
- 输出：单波段浮点栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出校正结果 |
| 波段号 | 整数 | 否 | `1` | 参与处理的波段 |
| dark_object_value | 浮点数 | 否 | 自动估计 | 手工指定暗像元值 |

## 算法内容

DOS 大气校正的核心思想是利用暗像元近似估计大气路径辐射，再把这部分偏移从原始影像中扣除。当前实现采用的是简化 DOS 流程，只针对单个处理波段工作，不引入完整的大气传输模型，更适合做基础预处理和轻量流程验证。

这里的处理结果本质上是“先减去暗背景，再按可用亮度范围重新缩放”的归一化结果。因此它更适合作为后续分析的相对校正输入，而不是严格的物理反射率产品。

## 处理流程

1. 打开输入栅格并读取指定波段。
2. 统计该波段的最小值和最大值。
3. 如果没有手动指定 `dark_object_value`，则直接使用当前波段最小值作为暗像元估计。
4. 从整幅波段中减去暗像元值。
5. 将减法后小于 0 的结果截断为 0，避免出现负辐射值。
6. 以 `最大值 - 暗像元值` 作为有效亮度范围，将结果缩放到近似 `0` 到 `1`。
7. 如果有效亮度范围过小，则直接输出全 0 结果。
8. 将校正后的结果写出为单波段浮点栅格。

## 输出说明

- 输出结果为非负浮点栅格
- 当亮暗差值可用时，结果会归一化到近似 `0` 到 `1`

## 结果解释

- 值越大，表示在扣除暗背景后该位置仍保持更高的相对亮度。
- 输出结果是简化归一化后的相对值，不等同于严格大气校正后的地表反射率。
- 如果输入影像整体动态范围很小，结果可能大面积接近 0。

## 适用场景

- 基础遥感预处理、快速联调和对比试验。
- 在缺少完整传感器定标与大气参数时，先做轻量暗像元校正。

## 推荐输出文件名

- `dos_correction_result.tif`

## 注意事项

- 当前实现为简化 DOS，适合基础流程验证与轻量处理
