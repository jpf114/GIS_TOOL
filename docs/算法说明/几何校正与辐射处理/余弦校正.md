# 几何校正与辐射处理 / 余弦校正

## 功能说明

根据坡度、坡向和太阳角度，对单波段影像执行余弦地形校正。

## 输入与输出

- 输入：
  - 输入影像
  - 坡度栅格
  - 坡向栅格
- 输出：
  - 校正结果栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出结果 |
| 波段号 | 整数 | 否 | `1` | 参与处理的波段 |
| slope_raster | 文件路径 | 是 | 无 | 坡度栅格，单位为度 |
| aspect_raster | 文件路径 | 是 | 无 | 坡向栅格，单位为度 |
| sun_zenith_deg | 浮点数 | 否 | `30` | 太阳天顶角，单位为度 |
| sun_azimuth_deg | 浮点数 | 否 | `180` | 太阳方位角，单位为度 |

## 输出说明

- 输出结果为单波段浮点栅格
- 入射角过小的位置会被写为 0

## 推荐输出文件名

- `cosine_correction_result.tif`
