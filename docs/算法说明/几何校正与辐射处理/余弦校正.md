# 几何校正与辐射处理 / 余弦校正

## 功能说明

根据坡度、坡向和太阳角度，对单波段影像执行余弦地形校正。

## 输入与输出

- 输入：
  - 输入影像
  - 坡度栅格
  - 坡向栅格
- 输出：
  - 校正结果栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出结果 |
| 波段号 | 整数 | 否 | `1` | 参与处理的波段 |
| slope_raster | 文件路径 | 是 | 无 | 坡度栅格，单位为度 |
| aspect_raster | 文件路径 | 是 | 无 | 坡向栅格，单位为度 |
| sun_zenith_deg | 浮点数 | 否 | `30` | 太阳天顶角，单位为度 |
| sun_azimuth_deg | 浮点数 | 否 | `180` | 太阳方位角，单位为度 |

## 算法内容

余弦校正是一种基础地形校正方法，用太阳入射角对地形阴坡、阳坡的亮度差异进行补偿。当前实现使用输入影像、坡度栅格、坡向栅格和太阳角度，逐像元计算地表与太阳之间的入射关系，再按余弦比值修正亮度。

它的核心目标是减弱由地形起伏带来的光照差异，而不是改变地物本身的光谱特性。

## 处理流程

1. 读取输入影像指定波段，以及坡度栅格和坡向栅格。
2. 检查三者尺寸是否一致，不一致则直接报错。
3. 将输入影像、坡度、坡向统一转为浮点格式。
4. 将太阳天顶角、太阳方位角、坡度和坡向从度转换为弧度。
5. 对每个像元计算当前地形条件下的太阳入射余弦值。
6. 用 `cos(太阳天顶角) / cos(入射角)` 作为校正系数，对原始像元值逐像元进行修正。
7. 对入射角余弦过小或不合理的位置，结果直接写为 0。
8. 将校正结果输出为单波段浮点栅格。

## 输出说明

- 输出结果为单波段浮点栅格
- 入射角过小的位置会被写为 0

## 结果解释

- 阳坡和阴坡因受光差异带来的亮度反差通常会被减弱。
- 如果某些位置几乎不受太阳直射，当前实现会直接输出 0，避免数值异常放大。
- 结果好坏高度依赖坡度、坡向以及太阳角度是否可信。

## 适用场景

- 山区遥感影像的基础地形光照校正。
- 为后续分类、指数或变化分析提供更均衡的亮度输入。

## 推荐输出文件名

- `cosine_correction_result.tif`

## 注意事项

- 结果质量高度依赖输入影像质量与参数设置。
- 建议先在样例数据上验证参数，再用于批量处理。
