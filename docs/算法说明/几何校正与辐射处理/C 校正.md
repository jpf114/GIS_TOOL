# 几何校正与辐射处理 / C 校正

## 功能说明

根据坡度、坡向、太阳角度和 `c_value`，对单波段影像执行 C 地形校正。

## 输入与输出

- 输入：
  - 输入影像
  - 坡度栅格
  - 坡向栅格
- 输出：
  - 校正结果栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出结果 |
| 波段号 | 整数 | 否 | `1` | 参与处理的波段 |
| slope_raster | 文件路径 | 是 | 无 | 坡度栅格，单位为度 |
| aspect_raster | 文件路径 | 是 | 无 | 坡向栅格，单位为度 |
| sun_zenith_deg | 浮点数 | 否 | `30` | 太阳天顶角 |
| sun_azimuth_deg | 浮点数 | 否 | `180` | 太阳方位角 |
| c_value | 浮点数 | 否 | `0.1` | C 校正系数 |

## 算法内容

C 校正是在余弦校正基础上引入常数项 `c_value` 的一种地形校正方式，目的是在入射角较小时抑制过强放大，降低纯余弦校正可能带来的不稳定。当前实现按输入影像、坡度、坡向和太阳角度逐像元计算校正比值，并把 `c_value` 同时加入分子和分母进行调节。

## 处理流程

1. 读取输入影像指定波段，以及坡度和坡向栅格。
2. 检查三者尺寸是否一致。
3. 将影像、坡度和坡向统一转为浮点格式。
4. 把太阳角度、坡度和坡向从度转换为弧度。
5. 对每个像元计算当前地形条件下的太阳入射余弦值。
6. 按 `((cos太阳天顶角 + c) / (cos入射角 + c))` 计算校正系数，并乘到原始像元值上。
7. 对分母过小的位置直接输出 0。
8. 将结果写出为单波段浮点栅格。

## 输出说明

- 输出结果为单波段浮点栅格
- 当分母过小时，结果会写为 0

## 结果解释

- `c_value` 越大，校正过程越不容易在弱入射区域产生剧烈放大。
- 与纯余弦校正相比，C 校正通常更稳一些，但也会降低校正的直接性。
- 参数是否合适，需要结合数据场景和样例对比判断。

## 适用场景

- 山区影像地形校正。
- 希望在补偿地形光照差异的同时，降低过校正风险的场景。

## 推荐输出文件名

- `c_correction_result.tif`

## 注意事项

- 结果质量高度依赖输入影像质量与参数设置。
- 建议先在样例数据上验证参数，再用于批量处理。
