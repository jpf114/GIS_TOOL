# 几何校正与辐射处理 / QUAC 大气校正

## 功能说明

按多波段亮暗百分位对输入影像执行简化快速大气校正。

## 输入与输出

- 输入：多波段栅格
- 输出：多波段浮点栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出校正结果 |
| dark_percentile | 浮点数 | 否 | `1.0` | 暗像元百分位 |
| bright_percentile | 浮点数 | 否 | `99.0` | 亮像元百分位 |

## 算法内容

当前实现的 QUAC 是一种简化快速大气校正思路：不依赖外部大气模型，而是按每个波段自身的亮暗分位值，先扣除暗背景，再按亮暗范围做归一化。它更接近“按波段做鲁棒亮度拉伸和归一化”的快速校正流程，适合轻量预处理和流程联调。

需要注意的是，这里并不是完整商业或科研口径的 QUAC 模型，而是当前项目中的简化实现版本。

## 处理流程

1. 打开输入多波段栅格。
2. 检查 `dark_percentile` 和 `bright_percentile` 的取值是否合理，并保证前者小于后者。
3. 按波段逐个读取输入数据。
4. 对每个波段统计指定暗百分位和亮百分位对应的像元值。
5. 从该波段所有像元中减去暗百分位值。
6. 将减法后小于 0 的结果截断为 0。
7. 以“亮百分位值 - 暗百分位值”作为当前波段的有效范围，对结果归一化。
8. 若有效范围过小，则该波段输出全 0。
9. 再把结果上限截断到 1，保证输出在 `0` 到 `1` 范围内。
10. 将所有处理后的波段合并写出为多波段浮点栅格。

## 输出说明

- 各波段会分别执行亮暗拉伸和归一化
- 输出值会被限制到 `0` 到 `1`

## 结果解释

- 输出值表示各波段在本波段亮暗范围内的相对校正结果。
- 由于每个波段独立按分位值归一化，结果更适合做后续相对分析，而不是严格物理反射率解释。
- `dark_percentile` 和 `bright_percentile` 的设置会直接影响校正力度与对极端值的敏感程度。

## 适用场景

- 多波段影像快速预处理。
- 缺少完整大气参数时的轻量校正与归一化。
- 为后续显示、指数计算或分类准备更平衡的输入数据。

## 推荐输出文件名

- `quac_correction_result.tif`

## 注意事项

- `dark_percentile` 必须小于 `bright_percentile`
- 当前实现为简化 QUAC，不依赖外部大气模型
