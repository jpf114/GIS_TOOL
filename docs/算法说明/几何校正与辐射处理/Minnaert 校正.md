# 几何校正与辐射处理 / Minnaert 校正

## 功能说明

根据坡度、坡向、太阳角度和 `minnaert_k`，对单波段影像执行 Minnaert 地形校正。

## 输入与输出

- 输入：
  - 输入影像
  - 坡度栅格
  - 坡向栅格
- 输出：
  - 校正结果栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入影像 |
| 输出文件 | 文件路径 | 是 | 无 | 输出结果 |
| 波段号 | 整数 | 否 | `1` | 参与处理的波段 |
| slope_raster | 文件路径 | 是 | 无 | 坡度栅格，单位为度 |
| aspect_raster | 文件路径 | 是 | 无 | 坡向栅格，单位为度 |
| sun_zenith_deg | 浮点数 | 否 | `30` | 太阳天顶角 |
| sun_azimuth_deg | 浮点数 | 否 | `180` | 太阳方位角 |
| minnaert_k | 浮点数 | 否 | `0.5` | Minnaert 系数 |

## 算法内容

Minnaert 校正可以看作对余弦校正的柔化版本，通过引入 `minnaert_k` 控制校正强度，使结果在复杂地表条件下不至于像纯余弦校正那样容易过补偿。当前实现根据坡度、坡向和太阳角度逐像元计算入射关系，再按 Minnaert 指数形式完成亮度修正。

## 处理流程

1. 读取输入影像指定波段，以及坡度和坡向栅格。
2. 检查三者尺寸是否一致。
3. 将输入影像、坡度和坡向统一转为浮点格式。
4. 把太阳角度、坡度和坡向从度转换为弧度。
5. 对每个像元计算当前地形条件下的太阳入射余弦值。
6. 用 `cos(太阳天顶角) / cos(入射角)` 作为基础比值，再以 `minnaert_k` 作为幂指数调整校正强度。
7. 对入射角余弦或太阳天顶角余弦过小的位置，结果直接写为 0。
8. 将校正结果输出为单波段浮点栅格。

## 输出说明

- 输出结果为单波段浮点栅格
- `minnaert_k` 越大，校正强度通常越接近余弦校正

## 结果解释

- 当 `minnaert_k` 较大时，结果更接近余弦校正。
- 当 `minnaert_k` 较小时，校正会更温和，适合避免部分区域被过度放大。
- 最终效果仍高度依赖坡度、坡向和太阳角度输入是否可靠。

## 适用场景

- 需要进行地形光照校正，但又希望比纯余弦校正更平缓的场景。
- 山区、多坡向区域的亮度均衡预处理。

## 推荐输出文件名

- `minnaert_correction_result.tif`

## 注意事项

- 结果质量高度依赖输入影像质量与参数设置。
- 建议先在样例数据上验证参数，再用于批量处理。
