# 投影转换 / 坐标转换

## 功能说明

将单个坐标点从源坐标系转换到目标坐标系，结果直接显示在执行结果中。

## 输入与输出

- 输入：坐标值与坐标系定义
- 输出：无文件输出，仅返回转换前后坐标

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 源坐标系 | 坐标系 | 否 | `EPSG:4326` | 输入点当前所处坐标系。 |
| 目标坐标系 | 坐标系 | 是 | 无 | 目标坐标系，例如 `EPSG:3857`。 |
| X 坐标 | 浮点数 | 否 | `0.0` | 待转换 X。 |
| Y 坐标 | 浮点数 | 否 | `0.0` | 待转换 Y。 |

## 算法内容

坐标转换用于把单个点坐标从源坐标系转换到目标坐标系。当前实现不会处理整幅栅格或整层矢量数据，而是只针对一对 `x,y` 坐标创建一次坐标变换并返回转换结果。

## 处理流程

1. 读取源坐标系、目标坐标系和输入坐标值。
2. 如果源坐标系未填写，则按 `EPSG:4326` 处理。
3. 解析源坐标系和目标坐标系定义。
4. 创建两者之间的坐标转换对象。
5. 对输入点执行一次坐标变换。
6. 将转换前后坐标以文本和元数据形式返回，不生成输出文件。

## 输出说明

- 本功能不生成输出文件。
- 转换结果直接显示在执行结果中，同时返回 `src_x`、`src_y`、`dst_x`、`dst_y` 等元数据。

## 结果解释

- `src_x`、`src_y` 是输入坐标。
- `dst_x`、`dst_y` 是转换到目标坐标系后的坐标值。
- 结果只代表当前这一个点，不代表整幅数据的重投影结果。

## 适用场景

- 快速验证坐标系转换是否正确。
- 在手工核对范围、定位点或参数时做单点换算。

## 注意事项

- 只有 `目标坐标系` 是硬性必填。
- 如果 `源坐标系` 留空，代码会按 `EPSG:4326` 处理。

## 填写示例

- 源坐标系：`EPSG:4326`
- 目标坐标系：`EPSG:3857`
- X 坐标：`120.123456`
- Y 坐标：`30.123456`

## 返回字段说明

- `src_x`、`src_y`：输入坐标
- `dst_x`、`dst_y`：转换后的目标坐标
