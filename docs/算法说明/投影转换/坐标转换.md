# 投影转换 / 坐标转换

## 功能说明

将单个坐标点从源坐标系转换到目标坐标系，结果直接显示在执行结果中。

## 输入与输出

- 输入：坐标值与坐标系定义
- 输出：无文件输出，仅返回转换前后坐标

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 源坐标系 | 坐标系 | 否 | `EPSG:4326` | 输入点当前所处坐标系。 |
| 目标坐标系 | 坐标系 | 是 | 无 | 目标坐标系，例如 `EPSG:3857`。 |
| X 坐标 | 浮点数 | 否 | `0.0` | 待转换 X。 |
| Y 坐标 | 浮点数 | 否 | `0.0` | 待转换 Y。 |

## 输出说明

- 输出结果为栅格文件。
- 建议优先使用 `.tif` 保存，便于后续在 CLI、GUI 或 GIS 软件中继续处理。

## 推荐输出文件名

- `坐标转换_result.tif`

## 注意事项

- 只有 `目标坐标系` 是硬性必填。
- 如果 `源坐标系` 留空，代码会按 `EPSG:4326` 处理。

## 填写示例

- 源坐标系：`EPSG:4326`
- 目标坐标系：`EPSG:3857`
- X 坐标：`120.123456`
- Y 坐标：`30.123456`

## 返回字段说明

- `src_x`、`src_y`：输入坐标
- `dst_x`、`dst_y`：转换后的目标坐标
