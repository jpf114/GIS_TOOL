# 投影转换 / 赋予坐标系

## 功能说明

直接给输入栅格写入坐标系定义，不做重采样，也不改变像元值与像素位置。

## 输入与输出

- 输入：栅格文件
- 输出：原地修改输入文件，不生成新文件

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 要写入坐标系的栅格文件。 |
| 坐标系 | 坐标系 | 是 | 无 | 例如 `EPSG:4326`，也可传 WKT。 |

## 注意事项

- 这是“写标签”，不是“重投影”。
- 如果原数据实际坐标系与填写内容不一致，后续空间分析会出错。
- 当前是原地修改输入文件，因此不会生成新的输出文件，也没有输出后缀可选。
