# 投影转换 / 重投影

## 功能说明

将输入数据重投影到目标坐标系。代码同时支持栅格和矢量输入，输出格式由输出文件后缀决定。

## 输入与输出

- 输入：
  - 栅格：常用 `.tif` `.tiff` `.img`
  - 矢量：常用 `.shp` `.gpkg` `.geojson` `.json` `.kml` `.csv`
- 输出：
  - 栅格建议使用 `.tif`
  - 矢量建议使用 `.gpkg`、`.geojson`、`.shp`
- 输出文件必填

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入栅格或矢量数据路径。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出路径，后缀决定输出格式。 |
| 目标坐标系 | 坐标系 | 是 | 无 | 例如 `EPSG:3857`，也可传 WKT。 |
| 源坐标系 | 坐标系 | 否 | 无 | 留空时优先使用输入数据自带坐标系。 |
| 重采样方式 | 枚举 | 否 | `nearest` | 栅格重投影时生效，可选 `nearest` `bilinear` `cubic` `cubicspline` `lanczos` `average` `mode`。 |

## 枚举值说明

### 重采样方式

- `nearest`：最近邻，速度快，适合分类栅格
- `bilinear`：双线性，适合连续型栅格
- `cubic`：三次卷积，结果更平滑
- `cubicspline`：三次样条
- `lanczos`：高质量重采样，速度相对更慢
- `average`：均值重采样
- `mode`：众数重采样，适合离散分类数据

## 处理逻辑

- 如果输入能按矢量打开，就走矢量重投影流程。
- 否则按栅格重投影处理。
- 矢量输出格式由输出后缀自动识别：
  - `.shp` -> `ESRI Shapefile`
  - `.gpkg` -> `GPKG`
  - `.kml` -> `KML`
  - `.csv` -> `CSV`
  - 其他默认按 `GeoJSON`

## 输出说明

- 输出为重投影后的实际数据文件。
- 栅格输出建议使用 `.tif`，矢量输出建议使用 `.gpkg`、`.geojson` 或 `.shp`。

## 推荐输出文件名

- `重投影_result.json`
- `重投影_result.csv`
- `重投影_result.gpkg`
- `重投影_result.geojson`
- `重投影_result.shp`
- `重投影_result.tif`

## 注意事项

- `目标坐标系` 不能为空。
- 栅格重投影时，输出一般应使用 `.tif`。
- 矢量写出前会删除同名旧输出，尤其是 Shapefile 会连带删同名前缀的侧车文件。
