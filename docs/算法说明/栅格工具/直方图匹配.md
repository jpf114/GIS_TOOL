# 栅格工具 / 直方图匹配

## 功能说明

将输入影像指定波段的灰度分布匹配到参考影像，使输出结果在整体亮度和对比度上更接近参考影像。

## 输入与输出

- 输入：待匹配栅格、参考栅格
- 输出：匹配后的栅格结果，建议使用 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 待执行匹配的输入栅格。 |
| 参考文件 | 文件路径 | 是 | 无 | 提供目标灰度分布的参考栅格。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出匹配结果。 |
| 波段号 | 整数 | 否 | `1` | 从 `1` 开始。 |

## 算法内容

直方图匹配用于让输入影像指定波段的灰度分布尽量接近参考影像。当前实现会先把输入和参考波段归一化到 8 位灰度，再分别统计两者直方图和累计分布函数，构建一张 256 级查找表，最后把输入影像映射到参考影像的灰度分布上。

## 处理流程

1. 打开输入栅格和参考栅格，并读取指定波段。
2. 如果参考影像尺寸与输入不一致，则先把参考影像重采样到输入尺寸。
3. 将输入波段和参考波段分别归一化到 `0-255` 的 8 位范围。
4. 统计两者的直方图和累计分布函数。
5. 根据累计分布函数关系构造 256 级灰度映射查找表。
6. 用该查找表把输入影像灰度映射到参考分布。
7. 将结果转换为浮点栅格并写出，同时保留输入影像的空间参考和地理变换。

## 注意事项

- 当前按单波段执行匹配，未指定时默认使用第 1 波段。
- 输入栅格与参考栅格应具有可比较的数据分布，极端异常值会影响匹配效果。
- 当前输出固定为 GTiff。

## 输出说明

- 输出结果为栅格文件。
- 建议优先使用 `.tif` 保存，便于后续在 CLI、GUI 或 GIS 软件中继续处理。

## 结果解释

- 输出影像的整体亮度和对比度分布会更接近参考影像。
- 该过程调整的是灰度分布，不保证空间纹理或绝对物理意义与参考完全一致。

## 适用场景

- 多景影像视觉一致化、亮度风格统一和显示前处理。
