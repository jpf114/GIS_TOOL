# 栅格工具 / 直方图匹配

## 功能说明

将输入影像指定波段的灰度分布匹配到参考影像，使输出结果在整体亮度和对比度上更接近参考影像。

## 输入与输出

- 输入：待匹配栅格、参考栅格
- 输出：匹配后的栅格结果，建议使用 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 待执行匹配的输入栅格。 |
| 参考文件 | 文件路径 | 是 | 无 | 提供目标灰度分布的参考栅格。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出匹配结果。 |
| 波段号 | 整数 | 否 | `1` | 从 `1` 开始。 |

## 注意事项

- 当前按单波段执行匹配，未指定时默认使用第 1 波段。
- 输入栅格与参考栅格应具有可比较的数据分布，极端异常值会影响匹配效果。
- 当前输出固定为 GTiff。

## 输出说明

- 输出结果为栅格文件。
- 建议优先使用 `.tif` 保存，便于后续在 CLI、GUI 或 GIS 软件中继续处理。

## 推荐输出文件名

```text
D:/data/scene_raster_histogram_match.tif
```
