# 栅格工具 / 伪彩色

## 功能说明

对单波段栅格应用颜色映射，输出 3 波段彩色结果。

## 输入与输出

- 输入：单波段或可指定单波段的栅格
- 输出：三波段彩色 GeoTIFF，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入栅格。 |
| 输出文件 | 文件路径 | 是 | 无 | 输出彩色结果。 |
| 波段号 | 整数 | 否 | `1` | 从 1 开始。 |
| 颜色映射 | 枚举 | 否 | `jet` | `jet` `viridis` `hot` `cool` `spring` `summer` `autumn` `winter` `bone` `hsv` `rainbow` `ocean`。 |

## 注意事项

- 当波段没有有效值域时会失败。
- 当前实现固定输出 GTiff。

## 枚举值说明

### 颜色映射

- `jet`：彩虹色
- `viridis`：Viridis
- `hot`：热度色带
- `cool`：冷色带
- `spring`：春季色带
- `summer`：夏季色带
- `autumn`：秋季色带
- `winter`：冬季色带
- `bone`：灰白骨架色带
- `hsv`：HSV 色环
- `rainbow`：彩虹
- `ocean`：海洋
