# 栅格工具 / 栅格信息

## 功能说明

查看栅格驱动、尺寸、范围、波段、统计值、颜色解释等信息。

## 输入与输出

- 输入：栅格文件
- 输出：无文件输出，仅返回文本结果

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入栅格。 |

## 返回内容

- 驱动类型
- 栅格宽高与波段数
- 原点、像元大小、旋转参数、范围
- CRS
- 概览层数
- 每个波段的类型、NoData、统计值、颜色解释

## 返回内容说明

- `Driver`：栅格驱动名称，如 `GTiff`
- `Size`：宽、高、波段数
- `GeoTransform`：原点、像元大小、旋转参数
- `Extent`：按 GeoTransform 推算的空间范围
- `CRS`：坐标参考系，若能识别会同时显示 authority 信息
- `Overviews`：第 1 波段已有的金字塔层数
- `Band n`：每个波段的详细信息
  - `Type`：波段数据类型
  - `NoData`：NoData 值
  - `Scale/Offset`：比例尺与偏移
  - `Min/Max/Mean/StdDev`：统计值
  - `Color`：颜色解释，如 `Red`、`Gray`

## 输出说明

- 输出结果为栅格文件。
- 建议优先使用 `.tif` 保存，便于后续在 CLI、GUI 或 GIS 软件中继续处理。

## 推荐输出文件名

- `栅格信息_result.tif`

## 注意事项

- 本功能只读，不生成输出文件。
- 如果原始数据没有统计信息，执行时会触发一次统计计算。
