# 栅格工具 / 栅格信息

## 功能说明

查看栅格驱动、尺寸、范围、波段、统计值、颜色解释等信息。

## 输入与输出

- 输入：栅格文件
- 输出：无文件输出，仅返回文本结果

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入栅格。 |

## 算法内容

栅格信息是一个只读检查动作，用于系统性查看栅格结构、空间参考、概览层和各波段统计信息。当前实现会打开输入栅格，逐项读取驱动、尺寸、GeoTransform、范围、CRS、概览层数以及每个波段的类型、NoData、统计值和颜色解释。

## 处理流程

1. 打开输入栅格。
2. 读取驱动名称、宽高和波段数。
3. 读取 GeoTransform，并计算空间范围。
4. 读取 CRS，如果能识别则同时返回 Authority 信息和 Pretty WKT。
5. 读取第 1 波段已有的概览层数量。
6. 遍历所有波段，提取类型、NoData、Scale/Offset、统计值和颜色解释。
7. 将上述内容整理为文本结果返回，不生成输出文件。

## 返回内容

- 驱动类型
- 栅格宽高与波段数
- 原点、像元大小、旋转参数、范围
- CRS
- 概览层数
- 每个波段的类型、NoData、统计值、颜色解释

## 返回内容说明

- `Driver`：栅格驱动名称，如 `GTiff`
- `Size`：宽、高、波段数
- `GeoTransform`：原点、像元大小、旋转参数
- `Extent`：按 GeoTransform 推算的空间范围
- `CRS`：坐标参考系，若能识别会同时显示 authority 信息
- `Overviews`：第 1 波段已有的金字塔层数
- `Band n`：每个波段的详细信息
  - `Type`：波段数据类型
  - `NoData`：NoData 值
  - `Scale/Offset`：比例尺与偏移
  - `Min/Max/Mean/StdDev`：统计值
  - `Color`：颜色解释，如 `Red`、`Gray`

## 结果解释

- 结果适合快速判断一个栅格是否具备完整空间参考、统计信息和可用波段结构。
- 如果原始数据缺少统计值，当前实现会在读取过程中触发一次统计计算。

## 适用场景

- 数据检查、问题定位、处理前核对和成果验收辅助。

## 输出说明

- 本功能不生成输出文件。
- 结果以文本形式返回，适合用于快速检查栅格结构、统计值和空间参考信息。

## 注意事项

- 本功能只读，不生成输出文件。
- 如果原始数据没有统计信息，执行时会触发一次统计计算。
