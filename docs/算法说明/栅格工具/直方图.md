# 栅格工具 / 直方图

## 功能说明

统计指定波段的直方图，可直接在结果中查看，也可导出为 JSON 文本文件。

## 输入与输出

- 输入：栅格文件
- 输出：
  - 可不写输出文件，仅在结果面板显示
  - 若导出文件，建议使用 `.json`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入栅格。 |
| 输出文件 | 文件路径 | 否 | 无 | 可选，导出直方图文本/JSON。 |
| 波段号 | 整数 | 否 | `1` | 从 1 开始。 |
| 分箱数 | 整数 | 否 | `256` | 必须大于 `0`。 |

## 算法内容

直方图用于统计指定波段数值在整个值域中的分布情况。当前实现先计算波段统计范围，再按给定分箱数把值域划分为多个区间，最后统计每个区间的像元数量，并以文本加 JSON 结构形式返回。

## 处理流程

1. 打开输入栅格并读取指定波段。
2. 统计该波段最小值和最大值。
3. 如果波段没有有效值域，则直接报错。
4. 按 `bins` 将值域划分为固定数量的分箱。
5. 调用 GDAL 直方图统计流程，得到每个分箱的计数。
6. 计算分箱宽度和每个分箱中心值。
7. 生成文本头部和 JSON 风格的直方图结果。
8. 如果填写了输出文件，则把结果写入文件；否则只在结果面板返回。

## 输出内容

- 文本头部会写统计范围、分箱宽度。
- 后面附带 JSON 结构的直方图数组。

## 输出文件示例

```json
{
  "histogram": [
    {
      "bin": 0,
      "center": 0.5,
      "count": 123
    }
  ],
  "total": 123
}
```

## 输出说明

- 本功能可只返回文本结果，也可把结果写出到文件。
- 结果内容包含统计范围、分箱宽度和 JSON 结构的直方图数组。

## 结果解释

- 每个 `bin` 代表一个值域区间。
- `center` 是该区间中心值。
- `count` 是落入该区间的像元数量。

## 适用场景

- 栅格值分布检查、阈值选择和数据质量分析。

## 推荐输出文件名

```text
D:/data/scene_raster_histogram.json
```

## 注意事项

- 建议先确认输入栅格可正常读取，且输出格式与目标场景匹配。
- 批量运行前可先用单个样例验证参数设置。
