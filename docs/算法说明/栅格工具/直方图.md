# 栅格工具 / 直方图

## 功能说明

统计指定波段的直方图，可直接在结果中查看，也可导出为 JSON 文本文件。

## 输入与输出

- 输入：栅格文件
- 输出：
  - 可不写输出文件，仅在结果面板显示
  - 若导出文件，建议使用 `.json`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入栅格。 |
| 输出文件 | 文件路径 | 否 | 无 | 可选，导出直方图文本/JSON。 |
| 波段号 | 整数 | 否 | `1` | 从 1 开始。 |
| 分箱数 | 整数 | 否 | `256` | 必须大于 `0`。 |

## 输出内容

- 文本头部会写统计范围、分箱宽度。
- 后面附带 JSON 结构的直方图数组。

## 输出文件示例

```json
{
  "histogram": [
    {
      "bin": 0,
      "center": 0.5,
      "count": 123
    }
  ],
  "total": 123
}
```

## 输出说明

- 输出结果为统计或检查结果文件。
- 可直接用于后续汇总、质检或二次分析。

## 推荐输出文件名

```text
D:/data/scene_raster_histogram.json
```

## 注意事项

- 建议先确认输入栅格可正常读取，且输出格式与目标场景匹配。
- 批量运行前可先用单个样例验证参数设置。
