# 栅格工具 / NoData 设置

## 功能说明

为单个波段或全部波段写入 NoData 值。

## 输入与输出

- 输入：栅格文件
- 输出：原地修改输入文件，不生成新文件

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入栅格。 |
| 波段号 | 整数 | 否 | `1` | 代码中 `<= 0` 表示对全部波段生效。 |
| NoData 值 | 浮点数 | 否 | `0.0` | 要写入的 NoData 数值。 |

## 输出说明

- 输出结果为栅格文件。
- 建议优先使用 `.tif` 保存，便于后续在 CLI、GUI 或 GIS 软件中继续处理。

## 推荐输出文件名

- `NoData_设置_result.tif`

## 注意事项

- GUI 默认值是 `1`，所以默认只改第 1 波段。
- 如果希望全部波段一起设置，需要传 `0` 或负数。
