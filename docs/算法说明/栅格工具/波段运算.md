# 栅格工具 / 波段运算

## 功能说明

按表达式对多波段栅格执行像元级计算，适合做波段加权、简单指数和线性组合。

## 输入与输出
- 输入：多波段栅格
- 输出：单波段结果栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入文件 | 文件路径 | 是 | 无 | 输入多波段栅格 |
| 输出文件 | 文件路径 | 是 | 无 | 输出运算结果 |
| 表达式 | 字符串 | 是 | 无 | 例如 `B1+B2`、`B1*0.5+B2*0.5` |

## 算法内容

波段运算用于对多波段栅格执行逐像元表达式计算。当前实现支持 `B1`、`B2` 这类波段占位符，以及基础四则运算和括号。它会先读取全部波段，再对每个像元代入表达式求值，生成单波段结果栅格。

## 处理流程

1. 打开输入多波段栅格。
2. 读取全部波段到内存。
3. 对每个像元构造一组 `B1/B2/...` 变量值。
4. 将表达式中的波段占位符替换成当前像元实际数值。
5. 按四则运算和括号规则求值，得到该像元输出值。
6. 对整幅影像逐像元重复计算。
7. 将结果写出为单波段浮点栅格，并保留原始栅格空间参考和地理变换。

## 输出说明

- 表达式中的波段占位符使用 `B1`、`B2`、`B3` 这种形式
- 当前支持基础四则运算和括号
- 输出保留输入栅格的空间参考和地理变换

## 结果解释

- 输出值含义完全由表达式决定。
- 如果表达式中出现除法且分母接近 0，当前实现会按 0 处理，避免结果爆掉。

## 适用场景

- 波段线性组合、简单指数试验和快速像元级栅格计算。

## 推荐输出文件名
- `band_math_result.tif`

## 注意事项

- 表达式引用的波段号必须落在输入栅格实际波段范围内
- 除法分母接近 0 时，当前实现会按 0 处理，避免结果溢出
