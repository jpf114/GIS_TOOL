# 算法说明总览

本文档按当前对外主功能分组整理，用于统一功能归类与算法入口说明。

说明：

- 当前文档侧会把 `raster_manage / raster_inspect / raster_render / raster_math` 合并整理为“栅格工具”
- GUI 左侧导航已将这 4 个插件合并为单一“栅格工具”主分组
- `classification` 主项统一按“分类统计”整理，监督分类动作仍归在该主功能下
- `spindex` 主项统一按“光谱指数”整理，NDVI 等指数动作不再归到“栅格工具”
- CLI 与底层插件体系仍以独立插件名暴露这些能力
- 后续主功能体系会继续向统一命名收口
